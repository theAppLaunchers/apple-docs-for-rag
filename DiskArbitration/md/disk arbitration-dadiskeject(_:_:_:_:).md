# DADiskEject(\_:\_:\_:\_:)

Ejects the specified disk object.

Mac Catalyst 13.1+macOS 10.4+

``` source
func DADiskEject(
    _ disk: DADisk,
    _ options: DADiskEjectOptions,
    _ callback: DADiskEjectCallback?,
    _ context: UnsafeMutableRawPointer?
)
```

## Parameters 

`disk`  

The disk object.

`options`  

The eject options.

`callback`  

The callback function to call once the ejection completes.

`context`  

The user-defined context parameter to pass to the callback function.
