# macOS Catalina 10.15.6 Release Notes

Update your apps to use new features, and test your apps against API changes.

## Overview

The macOS 10.15.4 SDK provides support to develop apps for Macs running macOS Catalina 10.15.6. The SDK comes bundled with Xcode 11.5, available from the Mac App Store. For information on the compatibility requirements for Xcode 11.5, see Xcode 11.5 Release Notes.

### General

There are no new release notes for this software update.
