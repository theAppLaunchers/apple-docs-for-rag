# macOS Monterey 12.2 Release Notes

Update your apps to use new features, and test your apps against API changes.

## Overview

The macOS 12.1 SDK provides support to develop apps for Mac computers running macOS Monterey 12.2. The SDK comes bundled with Xcode 13.2.1, available from the Mac App Store. For information on the compatibility requirements for Xcode 13.2.1, see Xcode 13.2.1 Release Notes.

### General

There are no new release notes for this software update.
