# macOS Mojave 10.14.3 Release Notes

Update your apps to use new features, and test your apps against API changes.

## Overview

The macOS 10.14.1 SDK provides support for developing apps for Macs running macOS Mojave 10.14.3. The SDK comes bundled with Xcode 10.1 available from the Mac App Store. For information on the compatibility requirements for Xcode 10.1, see Xcode Release Notes.

### General

There are no release notes for this software update.
