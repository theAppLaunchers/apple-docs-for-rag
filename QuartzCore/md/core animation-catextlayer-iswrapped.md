# isWrapped

Determines whether the text is wrapped to fit within the receiver’s bounds.

iOS 2.0+iPadOS 2.0+Mac Catalyst 13.1+macOS 10.5+tvOS 9.0+visionOS 1.0+

``` source
var isWrapped: Bool { get set }
```

## Discussion

Defaults to false.
