# init(appExtension:sceneID:)

Creates a new configuration object.

macOS 13.0+

``` source
init(
    appExtension: AppExtensionIdentity,
    sceneID: String
)
```

## Parameters 

`appExtension`  

The app extension for this configuration object.

`sceneID`  

The unique identifier for the scene that uses this configuration.
