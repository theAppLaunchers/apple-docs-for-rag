# endDate

The end time for the pedometer data.

iOS 8.0+iPadOS 8.0+Mac Catalyst 13.1+macOS 10.15+watchOS 2.0+

``` source
var endDate: Date { get }
```

## Discussion

The startDate and `endDate` represent the period of time over which the pedometer data was gathered.
