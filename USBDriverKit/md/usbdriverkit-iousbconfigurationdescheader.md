# IOUSBConfigurationDescHeader

The header of a configuration descriptor.

DriverKit 19.0+

``` source
struct IOUSBConfigurationDescHeader;
```

## Overview

Use this method to get the total length of the descriptor. See “USB 2.0 9.6.3: Configuration.”
