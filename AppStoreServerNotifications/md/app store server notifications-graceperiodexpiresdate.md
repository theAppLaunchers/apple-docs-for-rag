# gracePeriodExpiresDate

The time when the billing grace period for a subscription renewal expires.

App Store Server Notifications 2.0+

``` source
timestamp gracePeriodExpiresDate
```

## Discussion

For more information about billing grace periods, see Enable Billing Grace Period for auto-renewable subscriptions.
