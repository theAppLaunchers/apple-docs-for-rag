# AEStreamRef

An opaque data structure for storing stream-based descriptor data.

Mac Catalyst 13.0+macOS 10.0+

``` source
typealias AEStreamRef = OpaquePointer
```

## Discussion

You create `AEStreamRef` objects and manipulate their contents using the stream routines found in the section Apple Event Manager.
