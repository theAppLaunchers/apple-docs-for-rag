# kMDItemContentCreationDate

The creation date of an edited or optimized version of the song or composition.

macOS 10.4+

``` source
let kMDItemContentCreationDate: CFString!
```

## Discussion

This value is supplementary to kMDItemRecordingDate, which indicates the original recording date of the song or composition.
