# kMDItemRedEyeOnOff

Indicates if red-eye reduction was used to take the picture. A CFBoolean.

macOS 10.4+

``` source
let kMDItemRedEyeOnOff: CFString!
```

## Discussion

Possible values are 0 (no red-eye reduction mode or unknown) and 1 (red-eye reduction used).
