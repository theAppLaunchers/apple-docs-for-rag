# AEArrayType

Stores a value that specifies an array type.

Mac Catalyst 13.0+macOS 10.0+

``` source
typealias AEArrayType = Int8
```

## Discussion

You use this data type with the AEGetArray(_:_:_:_:_:_:_:) function and the AEPutArray(_:_:_:_:_:_:) function to specify an array type, using one of the constants from Data Array Constants.
