# AEStreamCloseDesc(\_:)

Marks the end of a descriptor in an `AEStreamRef`.

macOS 10.0+

``` source
func AEStreamCloseDesc(_ ref: AEStreamRef!) -> OSStatus
```

## Parameters 

`ref`  

An AEStreamRefcontaining the stream data.

## Return Value

A result code. See Result Codes.

## Discussion

Call this function to balance a preceding call to AEStreamOpenDesc(_:_:) or AEStreamOpenKeyDesc(_:_:_:). This function completes the definition of the `AEDesc`.
