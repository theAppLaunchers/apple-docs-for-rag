# kMDItemIsGeneralMIDISequence

Indicates whether the MIDI sequence contained in the file is setup for use with a General MIDI device. A CFBoolean.

macOS 10.4+

``` source
let kMDItemIsGeneralMIDISequence: CFString!
```
