# kMDItemMeteringMode

The metering mode used to take the image. A CFString.

macOS 10.4+

``` source
let kMDItemMeteringMode: CFString!
```

## Discussion

Possible values are: Unknown, Average, CenterWeightedAverage, Spot, MultiSpot, Pattern, and Partial.
