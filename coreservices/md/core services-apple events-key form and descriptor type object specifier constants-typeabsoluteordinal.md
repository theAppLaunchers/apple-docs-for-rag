# typeAbsoluteOrdinal

Specifies a descriptor whose data consists of one of the constants `kAEFirst`, `kAEMiddle`, `kAELast`, `kAEAny`, or `kAEAll`, which are described in AEDisposeToken(_:). Used with `formAbsolutePosition`.
