# kMDItemRecordingDate

The recording date of the song or composition.

macOS 10.4+

``` source
let kMDItemRecordingDate: CFString!
```

## Discussion

This value contains the original recording date of the song or composition and is supplementary to kMDItemContentCreationDate, which indicates the date of an edited or optimized version.
