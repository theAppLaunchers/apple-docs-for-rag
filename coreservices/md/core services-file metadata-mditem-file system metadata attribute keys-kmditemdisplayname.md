# kMDItemDisplayName

The localized version of the file name. A CFString.

macOS 10.4+

``` source
let kMDItemDisplayName: CFString!
```

## Discussion

This is the localized version of the LaunchServices call LSCopyDisplayNameForURL(_:_:)/LSCopyDisplayNameForRef(_:_:).
