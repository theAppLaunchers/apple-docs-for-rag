# ==(\_:\_:)

Returns a Boolean value that indicates whether the two add secure element pass errors are equal.

iOS 8.0+iPadOS 8.0+Mac Catalyst 14.5+macOS 10.10+visionOS 1.0+watchOS 2.0+Xcode 11.4+

``` source
static func == (lhs: PKAddSecureElementPassError, rhs: PKAddSecureElementPassError) -> Bool
```

## Parameters 

`lhs`  

An add secure element pass error to compare.

`rhs`  

Another add secure element pass error to compare.

## Return Value

`true` if the two errors are equal; otherwise, `false`.
