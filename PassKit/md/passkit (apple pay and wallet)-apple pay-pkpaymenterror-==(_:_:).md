# ==(\_:\_:)

Returns a Boolean value indicating whether two payment error objects are equal.

iOS 8.0+iPadOS 8.0+Mac Catalyst 14.5+macOS 10.10+visionOS 1.0+watchOS 2.0+Xcode 9.0+

``` source
static func == (lhs: PKPaymentError, rhs: PKPaymentError) -> Bool
```

## Parameters 

`lhs`  

A payment error object to compare.

`rhs`  

Another payment error object to compare.

## Return Value

`true` when the payment error objects are equal; otherwise, `false`.
