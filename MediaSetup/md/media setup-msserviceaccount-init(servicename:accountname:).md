# init(serviceName:accountName:)

Creates a new account.

iOS 14.0+iPadOS 14.0+Mac Catalyst 14.0+visionOS 1.0+

``` source
init(
    serviceName: String,
    accountName: String
)
```

## Parameters 

`serviceName`  

The name of the streaming media service.

`accountName`  

The user’s display name in the streaming media service.
