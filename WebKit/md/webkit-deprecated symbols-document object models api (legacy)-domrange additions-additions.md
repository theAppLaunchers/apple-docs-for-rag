# DOMRange Additions

Additions to the `DOMRange` class facilitate communication between the DOM API and WebKit and help convert web content into standard markup form.

## Topics

### Creating archives

var webArchive: WebArchive!

A web archive of the content in the range.

### Formatting content ranges

var markupString: String!

A string in markup format corresponding to the content in the range.
