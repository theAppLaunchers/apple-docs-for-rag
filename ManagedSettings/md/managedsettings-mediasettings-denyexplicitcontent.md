# denyExplicitContent

The metadata for the setting that denies explicit content.

iOS 15.0+iPadOS 15.0+Mac Catalyst 15.0+

``` source
static let denyExplicitContent: SettingMetadata
```

## Discussion

Use `denyExplicitContent` to access the metadata for denyExplicitContent. The default value is `false`.
