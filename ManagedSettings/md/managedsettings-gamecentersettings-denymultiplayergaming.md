# denyMultiplayerGaming

The metadata associated with the setting that prevents users from joining multiplayer games.

iOS 15.0+iPadOS 15.0+Mac Catalyst 15.0+

``` source
static let denyMultiplayerGaming: SettingMetadata
```

## Discussion

Use `denyMultiplayerGaming` to access the metadata for denyMultiplayerGaming. The default value is `false`.
