# denyAutoFill

The metadata associated with the setting that deactivates Safari’s AutoFill feature.

iOS 15.0+iPadOS 15.0+Mac Catalyst 15.0+

``` source
static let denyAutoFill: SettingMetadata
```

## Discussion

Use `denyAutoFill` to access the metadata for denyAutoFill. The default value is `false`.
