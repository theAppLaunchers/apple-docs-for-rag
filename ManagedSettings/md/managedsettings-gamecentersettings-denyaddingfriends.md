# denyAddingFriends

The metadata for the setting that prevents the user from adding friends in Game Center.

iOS 15.0+iPadOS 15.0+Mac Catalyst 15.0+

``` source
static let denyAddingFriends: SettingMetadata
```

## Discussion

Use `denyAddingFriends` to access the metadata for denyAddingFriends. The default value is `false`.
