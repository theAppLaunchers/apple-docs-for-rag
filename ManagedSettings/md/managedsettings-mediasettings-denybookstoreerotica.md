# denyBookstoreErotica

The metadata associated with the setting that denies access to content in the Books store categorized as erotica.

iOS 15.0+iPadOS 15.0+Mac Catalyst 15.0+

``` source
static let denyBookstoreErotica: SettingMetadata
```

## Discussion

Use `denyBookstoreErotica` to access the metadata for denyBookstoreErotica. The default value is `false`.
