# denySiri

The metadata for the constraint that configures access to Siri.

iOS 15.0+iPadOS 15.0+Mac Catalyst 15.0+

``` source
static let denySiri: SettingMetadata
```

## Discussion

Use `denySiri` to access the metadata for denySiri. The default value is `false`.
