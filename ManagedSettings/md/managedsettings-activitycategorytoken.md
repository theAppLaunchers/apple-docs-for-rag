# ActivityCategoryToken

A token that represents a category of app or website activity.

iOS 15.0+iPadOS 15.0+Mac Catalyst 15.0+

``` source
typealias ActivityCategoryToken = Token
```

## Discussion

Use `ActivityCategoryToken` to restrict and filter device applications without access to personal user data. FamilyActivitySelection provides tokens that devices within the same Family Sharing group can use to identify applications.
