# platform

The platform on which the customer consumed the in-app purchase.

App Store Server API 1.0+

``` source
int32 platform
```

## Possible Values 

`0`  

The platform is undeclared. Use this value to avoid providing information for this field.

`1`  

An Apple platform.

`2`  

Non-Apple platform.
