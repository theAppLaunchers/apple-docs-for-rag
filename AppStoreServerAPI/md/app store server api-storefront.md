# storefront

The three-letter code that represents the country or region associated with the App Store storefront of the purchase.

App Store Server API 1.8+

``` source
string storefront
```

## Mentioned in 

App Store Server API changelog

## Discussion

This property uses the ISO 3166-1 alpha-3 country code representation. This property is the same as the countryCode in StoreKit.
