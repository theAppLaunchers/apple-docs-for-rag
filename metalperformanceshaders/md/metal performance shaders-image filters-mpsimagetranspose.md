# MPSImageTranspose

A filter that transposes an image.

iOS 9.0+iPadOS 9.0+Mac Catalyst 13.0+macOS 10.13+tvOS 9.0+visionOS 1.0+

``` source
class MPSImageTranspose : MPSUnaryImageKernel
```

## Overview

An `MPSImageTranspose` filter applies a matrix transposition to the source image by exchanging its rows with its columns.
