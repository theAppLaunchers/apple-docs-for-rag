# init(\_:)

Creates an anchoring component with the given AR anchor.

iOS 13.0+iPadOS 13.0+Mac Catalyst 14.0+macOS 10.15+

``` source
init(_ anchor: ARAnchor)
```

## Parameters 

`anchor`  

An existing AR anchor to use.
