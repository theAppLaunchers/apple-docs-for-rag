# init(\_:)

Creates an anchoring component for a given target.

iOS 13.0+iPadOS 13.0+Mac Catalyst 14.0+macOS 10.15+visionOS

``` source
init(_ target: AnchoringComponent.Target)
```

## Parameters 

`target`  

The kind of real world object to target.
