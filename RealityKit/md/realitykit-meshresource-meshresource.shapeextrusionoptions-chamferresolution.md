# chamferResolution

Resolution of the chamfer curve.

iOS 18.0+iPadOS 18.0+Mac Catalyst 18.0+macOS 15.0+visionOS 2.0+

``` source
var chamferResolution: MeshResource.ShapeExtrusionOptions.CurveStrokeResolution
```

## Discussion

The default value is MeshResource.ShapeExtrusionOptions.CurveStrokeResolution.uniformSegmentsPerSpan(segmentCount:) with a segment count of 8.
