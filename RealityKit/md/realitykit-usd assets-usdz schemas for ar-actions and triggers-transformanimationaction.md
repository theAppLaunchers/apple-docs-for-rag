# TransformAnimationAction

An action that plays a transform animation.

## Overview

This action applies the transform animation defined by animation on the prims in affectedObjects.

### Declaration

```
class Preliminary_Action "TransformAnimationAction"
```

## Topics

### Properties

info:id

The action’s unique identifier.

affectedObjects

A list of prims that respond to the notification.

animation

A prim that contains a transform animation.
