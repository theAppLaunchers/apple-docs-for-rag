# init()

Creates an empty option set.

RealityKitSwiftiOSiPadOSMac CatalystmacOSvisionOS

``` source
init()
```

Available when `RawValue` conforms to `FixedWidthInteger`.

## Discussion

This initializer creates an option set with a raw value of zero.
