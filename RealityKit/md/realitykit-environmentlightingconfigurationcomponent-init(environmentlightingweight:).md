# init(environmentLightingWeight:)

Creates an environment-lighting configuration component.

iOS 18.0+iPadOS 18.0+Mac Catalyst 18.0+macOS 15.0+visionOS 2.0+

``` source
init(environmentLightingWeight: Float = 1.0)
```

## Parameters 

`environmentLightingWeight`  

A floating-point value in the range `[0.0, 1.0]`, where `0.0` indicates no environment-lighting contribution and `1.0` indicates full environment-lighting contribution.
