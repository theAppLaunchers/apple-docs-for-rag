# init(reverb:)

Creates a component from a reverberation setting.

iOS 18.0+iPadOS 18.0+Mac Catalyst 18.0+macOS 15.0+visionOS 2.0+

``` source
init(reverb: Reverb)
```

## Parameters 

`reverb`  

A setting the component uses to reverberate spatial audio.
