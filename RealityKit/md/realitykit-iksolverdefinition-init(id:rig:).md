# init(id:rig:)

Creates a solver definition for with a unique solver identifier and a rig.

iOS 18.0+iPadOS 18.0+Mac Catalyst 18.0+macOS 15.0+visionOS 2.0+

``` source
init(
    id: IKSolverDefinition.ID,
    rig: IKRig
)
```

## Parameters 

`id`  

The identifier the solver instance is using. Use a unique value for each ID in the resource.

`rig`  

The solver rig.
