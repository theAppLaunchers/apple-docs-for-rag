# init(tracking:)

Creates a configuration with a set of anchor capabilities.

visionOS 2.0+

``` source
init(tracking capabilities: Set)
```

## Parameters 

`capabilities`  

The set of anchor capabilities to run with a SpatialTrackingSession.
