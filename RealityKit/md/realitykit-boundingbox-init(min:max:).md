# init(min:max:)

Creates a bounding box with the given settings.

iOS 13.0+iPadOS 13.0+Mac Catalyst 13.0+macOS 10.15+visionOS

``` source
init(
    min: SIMD3,
    max: SIMD3
)
```

## Parameters 

`min`  

The position of the minimum corner of the box.

`max`  

The position of the maximum corner of the box.
