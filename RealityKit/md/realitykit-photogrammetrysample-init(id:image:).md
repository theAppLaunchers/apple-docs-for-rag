# init(id:image:)

Creates a new sample.

iOS 17.0+iPadOS 17.0+Mac Catalyst 15.0+macOS 12.0+

``` source
init(
    id: Int,
    image: CVPixelBuffer
)
```

## Parameters 

`id`  

A unique identifier for the sample.

`image`  

The image data in one of the following formats:

- kCVPixelFormatType_32BGRA

  - kCVPixelFormatType_32ARGB
