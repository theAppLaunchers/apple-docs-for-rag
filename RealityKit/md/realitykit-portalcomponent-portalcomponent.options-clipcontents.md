# clipContents

An option that enables the clipping feature.

iOS 18.0+iPadOS 18.0+Mac Catalyst 18.0+macOS 15.0+visionOS 2.0+

``` source
static let clipContents: PortalComponent.Options
```

## Discussion

If this option is in the option set, the user-defined geometry for clippingMode clips the contents within the portal world.
