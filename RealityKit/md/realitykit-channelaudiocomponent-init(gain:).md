# init(gain:)

Configure the behavior of a channel audio source.

iOS 18.0+iPadOS 18.0+Mac Catalyst 18.0+macOS 15.0+visionOS 1.0+

``` source
init(gain: Audio.Decibel = .zero)
```

## Parameters 

`gain`  

The overall level for all sounds emitted from an entity.
