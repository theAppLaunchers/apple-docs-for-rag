# vmnet_estimated_packets_available_key

Mac Catalyst 13.0+macOS 10.10+

``` source
var vmnet_estimated_packets_available_key: UnsafePointer
```

## Discussion

The estimated number of packets available to be read.

This key is used for the `vmnet` event.

The value for this key is of type doc://com.apple.documentation/documentation/xpc/xpc_type_uint64.
