# GLKVector3Make(\_:\_:\_:)

Returns a new three-component vector created from individual component values.

iOS 5.0+iPadOS 5.0+macOS 10.8+tvOS 9.0+

``` source
func GLKVector3Make(
    _ x: Float,
    _ y: Float,
    _ z: Float
) -> GLKVector3
```

## Parameters 

`x`  

The first component.

`y`  

The second component.

`z`  

The third component.

## Return Value

An initialized vector.
