# GLKVector2Distance(\_:\_:)

Returns the distance between two points.

iOS 5.0+iPadOS 5.0+macOS 10.8+tvOS 9.0+

``` source
func GLKVector2Distance(
    _ vectorStart: GLKVector2,
    _ vectorEnd: GLKVector2
) -> Float
```

## Parameters 

`vectorStart`  

A vector providing the coordinates of the starting point.

`vectorEnd`  

A vector providing the coordinates of the ending point.

## Return Value

The distance between the two points.
