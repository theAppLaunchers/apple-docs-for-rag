# setAttributedText(\_:)

Sets the styled text displayed by the text field.

watchOS 6.0+

``` source
func setAttributedText(_ attributedText: NSAttributedString?)
```

## Parameters 

`attributedText`  

The styled text displayed in the text field. Specifying `nil` clears the current text.
