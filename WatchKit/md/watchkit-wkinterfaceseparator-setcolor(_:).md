# setColor(\_:)

Sets the color of the separator bar.

watchOS 2.0+

``` source
func setColor(_ color: UIColor?)
```

## Parameters 

`color`  

The custom color to apply to the separator. Specifying `nil` removes the custom color and returns the separator to the color specified in the storyboard file. The default color is white.
