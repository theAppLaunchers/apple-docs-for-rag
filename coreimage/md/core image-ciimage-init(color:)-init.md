# init(color:)

Initializes an image of infinite extent whose entire content is the specified color.

iOS 5.0+iPadOS 5.0+Mac Catalyst 13.1+macOS 10.4+tvOS 9.0+visionOS 1.0+

``` source
init(color: CIColor)
```

## Parameters 

`color`  

A color object.

## Return Value

The initialized image object.
