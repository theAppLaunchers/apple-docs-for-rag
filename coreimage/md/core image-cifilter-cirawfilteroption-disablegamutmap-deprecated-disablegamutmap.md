# disableGamutMap

A key for an NSNumber object containing a Boolean value that determines whether or not to disable gamut mapping.

iOS 10.0–15.0DeprecatediPadOS 10.0–15.0DeprecatedMac Catalyst 13.1–15.0DeprecatedmacOS 10.12–12.0DeprecatedtvOS 10.0–15.0DeprecatedvisionOS 1.0–1.0Deprecated

``` source
static let disableGamutMap: CIRAWFilterOption
```

## Discussion

A value of `true` disables gamut mapping. The default is `false`.
