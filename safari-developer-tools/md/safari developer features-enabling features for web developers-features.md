# Enabling features for web developers

Enable features and settings for web developers in Safari.

## 

Safari on macOS has several tools for web developers, including the Develop menu, Web Inspector, and WebDriver, which are turned off by default. To enable these tools:

1.  From the menu bar, choose **Safari**.

2.  Select **Settings…** (⌘,).

3.  Go to the **Advanced** pane.

4.  Check the **Show features for web developers** checkbox.
