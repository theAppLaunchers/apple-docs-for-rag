# kQLPreviewPDFPagesWithThumbnailsOnLeftStyle

The content of the PDF appears with thumbnails of all pages on the left side of the current page’s content.

macOS 10.5+

``` source
var kQLPreviewPDFPagesWithThumbnailsOnLeftStyle: QLPreviewPDFStyle { get }
```
