# kQLPreviewContentIDScheme

The content ID URL scheme.

macOS 10.0–12.0Deprecated

``` source
let kQLPreviewContentIDScheme: CFString!
```

Deprecated

Use a QLPreviewingController in a Preview Extension to provide previews for your file types.

## Discussion

The `cid` URL scheme permits the HTML of a mail message to refer to the images or other data included in the message. For more information, read RFC 2111.
