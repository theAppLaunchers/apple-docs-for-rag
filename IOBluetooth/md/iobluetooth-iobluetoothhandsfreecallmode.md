# IOBluetoothHandsFreeCallMode

The type of call data.

macOS

``` source
let IOBluetoothHandsFreeCallMode: String
```

## Discussion

The string contains a single digit with one of the following values:

`“0”`  
A voice call.

`“1”`  
A data call.

`“2”`  
A fax call.
