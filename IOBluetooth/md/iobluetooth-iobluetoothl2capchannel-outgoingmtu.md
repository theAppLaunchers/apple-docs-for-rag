# outgoingMTU

Returns the current outgoing MTU for the L2CAP channel.

macOS

``` source
var outgoingMTU: BluetoothL2CAPMTU { get }
```

## Discussion

The outgoing MTU represents the maximum L2CAP packet size for packets being sent to the remote device.
