# Apple Pay on the Web Version 12 Release Notes

The version of Apple Pay available in macOS 12 and iOS 15.

## Overview

New features include:

- Added the properties total, additionalLineItems, and additionalShippingMethods to ApplePayModifier to support the W3C Payment Request API.

- Added dateComponentsRange to ApplePayShippingMethod to provide estimated arrival dates for shipping methods.

- Added supportsCouponCode and couponCode to ApplePayRequest to support the W3C Payment Request API.

- Created ApplePayCouponCodeDetails to encapsulate the information within methodDetails from PaymentMethodChangeEvent.

- Created ApplePayCouponCodeChangedEvent.

- Added completeCouponCodeChange to ApplePaySession.

- Created ApplePayCouponCodeUpdate.

- Added shippingContactEditingMode to ApplePayRequest to support the W3C Payment Request API which allows you to define whether a user can edit a shipping address.

- Added supportsCouponCode, couponCode, and shippingContactEditingMode to ApplePayPaymentRequest.
