# locale

The language and region used for the displayed Apple Pay button.

Safari Desktop 10.0+Safari Mobile 10.0+

``` source
attribute ApplePayButtonLocale locale;
```

## Discussion

For a list of button locales, see ApplePayButtonLocale.
