# type

The kind of Apple Pay button, such as a button for purchasing a subscription.

Safari Desktop 10.0+Safari Mobile 10.0+

``` source
attribute ApplePayButtonType type;
```

## Discussion

For a list of button types, see ApplePayButtonType.
