# hours

A number that represents an hour.

Safari Desktop 10.0+Safari Mobile 10.0+

``` source
long hours;
```

## Discussion

Hours of the day are 0-based. For example, to represent the time 06:00, use `6`, and to represent midnight, use `0`.
