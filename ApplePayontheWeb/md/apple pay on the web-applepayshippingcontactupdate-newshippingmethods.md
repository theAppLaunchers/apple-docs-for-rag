# newShippingMethods

A list of shipping methods that are available to the updated shipping contact.

Safari Desktop 10.0+Safari Mobile 10.0+

``` source
sequence  newShippingMethods;
```

## Discussion

Supply a list of shipping methods that apply to the updated shipping contact, if the shipping methods changed. See ApplePayShippingMethod.
