# init(domain:error:)

Creates a Core Telephony error instance with the given values.

iOS 4.0+iPadOS 4.0+Mac Catalyst 14.0+macOS 10.10+

``` source
init(
    domain: Int32,
    error: Int32
)
```

## Parameters 

`domain`  

The domain of the error. See Error Domains for possible values of this parameter.

`error`  

The code of the error.
