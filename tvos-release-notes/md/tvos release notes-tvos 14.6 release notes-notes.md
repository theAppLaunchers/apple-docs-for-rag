# tvOS 14.6 Release Notes

Update your apps to use new features, and test your apps against API changes.

## Overview

To develop apps for tvOS 14.6, build your apps against the tvOS 14.5 SDK or later. The SDK comes bundled with Xcode 12.5, available from the Mac App Store. For information on the compatibility requirements for Xcode 12.5, see Xcode 12.5 Release Notes.

### General

There are no new release notes for this software update.
