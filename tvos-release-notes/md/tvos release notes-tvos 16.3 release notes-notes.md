# tvOS 16.3 Release Notes

Update your apps to use new features, and test your apps against API changes.

## Overview

The tvOS 16.1 SDK provides support to develop tvOS apps for Apple TV devices running tvOS 16.3. The SDK comes bundled with Xcode 14.2, available from the Mac App Store. For information on the compatibility requirements for Xcode 14.2, see Xcode 14.2 Release Notes.

### General

There are no new release notes for this software update.
