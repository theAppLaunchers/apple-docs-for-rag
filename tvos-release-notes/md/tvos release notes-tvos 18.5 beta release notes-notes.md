# tvOS 18.5 Beta Release Notes

Update your apps to use new features, and test your apps against API changes.

## Overview

The tvOS 18.5 SDK provides support to develop tvOS apps for Apple TV devices running tvOS 18.5 beta. The SDK comes bundled with Xcode 16.4, available from the Mac App Store. For information on the compatibility requirements for Xcode 16.4, see Xcode 16.4 Release Notes.
