# tvOS 18.3 Release Notes

Update your apps to use new features, and test your apps against API changes.

## Overview

The tvOS 18.2 SDK provides support to develop tvOS apps for Apple TV devices running tvOS 18.3. The SDK comes bundled with Xcode 16.2, available from the Mac App Store. For information on the compatibility requirements for Xcode 16.2, see Xcode 16.2 Release Notes.

### General

There are no new release notes for this software update.
