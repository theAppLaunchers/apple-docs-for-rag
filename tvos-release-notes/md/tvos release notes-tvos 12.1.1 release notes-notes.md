# tvOS 12.1.1 Release Notes

Update your apps to use new features, and test your apps against API changes.

## Overview

The tvOS 12.1 SDK provides support for developing tvOS apps for Apple TV devices running tvOS 12.1.1. The SDK comes bundled with Xcode 10.1 available from the Mac App Store. For information on the compatibility requirements for Xcode 10.1, see Xcode Release Notes.
