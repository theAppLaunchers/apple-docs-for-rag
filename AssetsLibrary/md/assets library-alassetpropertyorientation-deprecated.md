# ALAssetPropertyOrientation

The key to retrieve the orientation of the asset.

iOS 4.0–9.0DeprecatediPadOS 4.0–9.0DeprecatedMac Catalyst 13.1–13.1Deprecated

``` source
let ALAssetPropertyOrientation: String
```

Deprecated

Use the orientation of the UIImage returned for a PHAsset via the PHImageManager from the Photos framework instead

## Discussion

The corresponding value is an NSNumber object containing an asset’s orientation as described by the TIFF format.
