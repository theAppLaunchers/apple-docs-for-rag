# Revoke a Certificate

Revoke a lost, stolen, compromised, or expiring signing certificate.

## URL

``` source
DELETE https://api.enterprise.developer.apple.com/v1/certificates/{id}
```

## Path Parameters

`id`

`string`

 (Required)
