# Enterprise Program API 1.0 release notes

The initial version of the Enterprise Program API.

## Overview

This REST API provides resources that enable you to automate actions you take in developer.apple.com

### New features

- The API supports provisioning, including managing bundle IDs with the Bundle IDs and Bundle ID Capabilities resources, managing pass type IDs with Pass Type Ids resource, managing certificates with the Certificates resource, registering test devices with the Devices resource, and managing provisioning profiles with Profiles.

- Manage your users and their access with the Users resources. Invite users to join your team with the User Invitations resource.
