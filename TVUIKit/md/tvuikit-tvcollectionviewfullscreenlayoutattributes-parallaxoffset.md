# parallaxOffset

The number of points by which to shift the background from the center when moving focus.

tvOS 13.0+

``` source
@MainActor
var parallaxOffset: CGFloat { get set }
```

## Discussion

A larger offset value creates a more dramatic parallax effect, and a smaller offset value creates a less dramatic effect.
