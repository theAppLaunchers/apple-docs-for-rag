# displayedKeys

The keys to be displayed when a contact is expanded.

macOS 10.11+

``` source
var displayedKeys: [String] { get set }
```

## Discussion

If no keys are provided, the picker selects contacts instead of values. For a list of possible keys, see Contact Keys.
