# materialFace

The surface of an object.

iOS 9.0+iPadOS 9.0+Mac Catalyst 13.1+macOS 10.11+tvOS 9.0+visionOS 1.0+

``` source
var materialFace: MDLMaterialFace { get set }
```

## Discussion

Determines if the material is front-facing, back-facing, or double-sided. The default case of a materialFace is MDLMaterialFace.front. For more information, see MDLMaterialFace.
