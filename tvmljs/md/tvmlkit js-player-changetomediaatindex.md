# changeToMediaAtIndex

Immediately begins playing the media item at the specified index in the playlist.

tvOS 10.0+

``` source
void changeToMediaAtIndex(
    in int index
);
```

## Parameters 

`index`  

An integer indicating a position in the playlist.
