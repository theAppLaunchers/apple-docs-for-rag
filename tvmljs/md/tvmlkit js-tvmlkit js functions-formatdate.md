# formatDate

Formats the given date into the given format.

tvOS 9.0+

``` source
String formatDate(
    in Date date, 
    in String format
);
```

## Parameters 

`date`  

The date value to format.

`format`  

A string that specifies the desired format. For a list of possible formats, see DateFormatter.

## Return Value

A string containing the formatted date.
