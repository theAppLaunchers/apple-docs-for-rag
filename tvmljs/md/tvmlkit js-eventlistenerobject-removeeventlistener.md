# removeEventListener

Removes the designated event listener.

tvOS 9.0+

``` source
void removeEventListener(
    in String type, 
    in Object listener
);
```

## Parameters 

`type`  

The developer defined name of the event type to remove.

`listener`  

The listener object to be removed. This object is typically a function.
