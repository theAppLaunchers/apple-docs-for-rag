# canOpenURL

Determines if a deep-link to another app can be opened.

tvOS 10.0+

``` source
Boolean canOpenURL(
    in String url
);
```

## Parameters 

`url`  

The deep-link URL being verified.

## Return Value

Returns `YES` if the app can be opened.
