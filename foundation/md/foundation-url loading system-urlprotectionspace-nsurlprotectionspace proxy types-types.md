# NSURLProtectionSpace proxy types

These constants describe the supported proxy types used in init(proxyHost:port:type:realm:authenticationMethod:) and returned by proxyType.

## Topics

### Proxy types

let NSURLProtectionSpaceHTTPProxy: String

The proxy type for HTTP proxies.

let NSURLProtectionSpaceHTTPSProxy: String

The proxy type for HTTPS proxies.

let NSURLProtectionSpaceFTPProxy: String

The proxy type for FTP proxies.

Deprecated

let NSURLProtectionSpaceSOCKSProxy: String

The proxy type for SOCKS proxies.
