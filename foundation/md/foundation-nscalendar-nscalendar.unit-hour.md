# hour

Identifier for the hour unit.

iOS 2.0+iPadOS 2.0+Mac Catalyst 13.0+macOS 10.0+tvOS 9.0+visionOS 1.0+watchOS 2.0+


## Discussion

The corresponding value is an `NSInteger`. Equal to `kCFCalendarUnitHour`.
