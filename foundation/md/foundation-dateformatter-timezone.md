# timeZone

The time zone for the receiver.

iOS 2.0+iPadOS 2.0+Mac Catalyst 13.0+macOS 10.0+tvOS 9.0+visionOS 1.0+watchOS 2.0+

``` source
var timeZone: TimeZone! { get set }
```

## Discussion

If unspecified, the system time zone is used.
