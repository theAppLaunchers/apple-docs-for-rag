# init()

Creates an empty operating system version.

iOS 2.0+iPadOS 2.0+Mac Catalyst 13.0+macOS 10.0+tvOS 9.0+visionOS 1.0+watchOS 2.0+

``` source
init()
```

## Discussion

After initialization, the majorVersion, minorVersion, and patchVersion are all `0`.
