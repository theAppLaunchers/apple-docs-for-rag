# Byte Order

Recognize constants that specify a byte-order format.

## Overview

These constants are returned by NSHostByteOrder().

## Topics

### Constants

var NS_BigEndian: Int

The byte order is big endian.

var NS_LittleEndian: Int

The byte order is little endian.

var NS_UnknownByteOrder: Int

The byte order is unknown.
