# init(value:unit:)

Create a `Measurement` given a specified value and unit.

iOS 10.0+iPadOS 10.0+Mac Catalyst 10.0+macOS 10.12+tvOS 10.0+visionOS 1.0+watchOS 3.0+

``` source
init(
    value: Double,
    unit: UnitType
)
```

## Parameters 

`value`  

The double-precision floating-point measurement value.

`unit`  

The unit of measure.
