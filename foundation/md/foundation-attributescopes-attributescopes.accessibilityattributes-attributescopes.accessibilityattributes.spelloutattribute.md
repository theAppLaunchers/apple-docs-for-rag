# AttributeScopes.AccessibilityAttributes.SpellOutAttribute

An attribute to define if each character should be spoken separately.

FoundationAccessibilityiOS 15.0+iPadOS 15.0+Mac Catalyst 15.0+macOS 12.0+tvOS 15.0+visionOS 1.0+watchOS 8.0+


## Overview

For example, a number representing a series of digits, like 25, spoken as “two-five” rather than “twenty-five”. Another usecase is acronyms, like APPL, spoken as “A-P-P-L”
