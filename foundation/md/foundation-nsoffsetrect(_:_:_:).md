# NSOffsetRect(\_:\_:\_:)

Offsets the rectangle by the specified amount.

Mac Catalyst 13.0+macOS 10.0+

``` source
func NSOffsetRect(
    _ aRect: NSRect,
    _ dX: Double,
    _ dY: Double
) -> NSRect
```

## Return Value

A copy of `aRect`, with its location shifted by `dX` along the x axis and by `dY` along the y axis.
