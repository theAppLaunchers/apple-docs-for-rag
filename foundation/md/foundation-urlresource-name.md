# name

The name of the resource in the bundle.

iOS 16.0+iPadOS 16.0+Mac Catalyst 16.0+macOS 13.0+tvOS 16.0+visionOS 1.0+watchOS 9.0+

``` source
let name: String
```

## Discussion

The name property specifies both the resource name and its file extension.
