# throughputKey

A key with a corresponding value that indicates the speed of data processing, in bytes per second.

iOS 7.0+iPadOS 7.0+Mac Catalyst 13.1+macOS 10.9+tvOS 9.0+visionOS 1.0+watchOS 2.0+

``` source
static let throughputKey: ProgressUserInfoKey
```

## Discussion

If present, the autogenerated localizedAdditionalDescription uses the value to provide a string that presents more information.
