# markedClauseSegment

The index of the marked clause segment.

macOS 10.0+

``` source
static let markedClauseSegment: NSAttributedString.Key
```

## Discussion

The value of this attribute is an NSNumber object containing an integer, as an index in marked text indicating clause segments.
