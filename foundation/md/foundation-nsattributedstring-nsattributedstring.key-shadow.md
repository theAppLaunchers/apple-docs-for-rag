# shadow

The shadow of the text.

iOS 6.0+iPadOS 6.0+Mac Catalyst 13.1+macOS 10.0+tvOS 9.0+visionOS 1.0+watchOS 2.0+

``` source
static let shadow: NSAttributedString.Key
```

## Discussion

The value of this attribute is an NSShadow object. The default value of this property is `nil`.
