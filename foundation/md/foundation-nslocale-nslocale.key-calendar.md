# calendar

The calendar associated with the locale.

iOS 2.0+iPadOS 2.0+Mac Catalyst 13.0+macOS 10.0+tvOS 9.0+visionOS 1.0+watchOS 2.0+

``` source
static let calendar: NSLocale.Key
```

## Discussion

The corresponding value is an `NSCalendar` object.
