# short

The short representation of a standalone weekday.

iOS 15.0+iPadOS 15.0+Mac Catalyst 15.0+macOS 12.0+tvOS 15.0+visionOS 1.0+watchOS 8.0+

``` source
static var short: Date.FormatStyle.Symbol.StandaloneWeekday { get }
```

## Discussion

This custom format style conveys a short representation of a standalone weekday like `Mo` for Monday. For an even shorter representation, see narrow.
