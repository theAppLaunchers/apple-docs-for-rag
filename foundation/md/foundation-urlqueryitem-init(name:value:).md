# init(name:value:)

Creates a new query item with the name and value you specify.

iOS 8.0+iPadOS 8.0+Mac Catalyst 8.0+macOS 10.10+tvOS 9.0+visionOS 1.0+watchOS 2.0+

``` source
init(
    name: String,
    value: String?
)
```

## Parameters 

`name`  

The name for the query item.

`value`  

The value for the query item.
