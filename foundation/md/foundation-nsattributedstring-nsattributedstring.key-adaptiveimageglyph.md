# adaptiveImageGlyph

The adaptive image glyph for the text.

iOS 18.0+iPadOS 18.0+Mac Catalyst 18.0+macOS 15.0+tvOS 18.0+visionOS 2.0+watchOS 11.0+

``` source
static let adaptiveImageGlyph: NSAttributedString.Key
```

## Discussion

The value of this attribute is an NSAdaptiveImageGlyph object. The default value of this property is `nil`, which indicates there is no adaptive image glyph.
