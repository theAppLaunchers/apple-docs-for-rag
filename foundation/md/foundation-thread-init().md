# init()

Returns an initialized `NSThread` object.

iOS 2.0+iPadOS 2.0+Mac Catalyst 13.1+macOS 10.5+tvOS 9.0+visionOS 1.0+watchOS 2.0+

``` source
init()
```

## Return Value

An initialized `NSThread` object.

## Discussion

This is the designated initializer for `NSThread`.
