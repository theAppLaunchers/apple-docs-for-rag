# AttributeScopes.AccessibilityAttributes.TextualContextAttribute.TextualContext

Textual context that assistive technologies can use to improve the presentation of spoken text.

iOS 15.0+iPadOS 15.0+Mac Catalyst 15.0+macOS 12.0+tvOS 15.0+visionOS 1.0+watchOS 8.0+

``` source
enum TextualContext
```

## Topics

### Enumeration Cases

case console

Text used for input, like in the Terminal app.

case fileSystem

Text used for a file name or path, like in the Files app.

case messaging

Text used in a message, like in the Messages app.

case narrative

Text of long form reading content, like in the Books or News app.

case plain

Text that has no specific context.

case sourceCode

Text used in source code, like in Swift Playgrounds.

case spreadsheet

Text used in a grid of data, like in the Numbers app.

case wordProcessing

Text used in a document, like in the Pages app.
