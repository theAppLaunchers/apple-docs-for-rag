# init(strokes:)

Creates a drawing object with the strokes you supply.

iOS 14.0+iPadOS 14.0+Mac Catalyst 14.0+macOS 11.0+visionOS 1.0+

``` source
convenience init(strokes: [PKStroke])
```

## Parameters 

`strokes`  

An array of PKStrokeReference elements.
