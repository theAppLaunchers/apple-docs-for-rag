# kCTFontDownloadedAttribute

The download state.

iOS 7.0+iPadOS 7.0+Mac Catalyst 13.1+macOS 10.12+tvOS 9.0+visionOS 1.0+watchOS 2.0+

``` source
let kCTFontDownloadedAttribute: CFString
```

## Discussion

The value associated with this key is a doc://com.apple.documentation/documentation/corefoundation/cfboolean-s0p. If it is kCFBooleanTrue, corresponding Font Asset has been downloaded.

Note

It may still be necessary to call appropriate API in order to use the font in the Font Asset.
