# CTFontCopyNameForGlyph(\_:\_:)

Retrieves the name for the specified glyph.

iOS 6.0+iPadOS 6.0+Mac Catalyst 13.1+macOS 10.8+tvOS 9.0+visionOS 1.0+watchOS 2.0+

``` source
func CTFontCopyNameForGlyph(
    _ font: CTFont,
    _ glyph: CGGlyph
) -> CFString?
```

## Parameters 

`font`  

The font reference.

`glyph`  

The glyph.

## Return Value

The glyph name as a CFString, or `NULL` if the glyph is invalid.
