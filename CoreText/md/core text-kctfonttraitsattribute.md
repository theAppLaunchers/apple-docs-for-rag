# kCTFontTraitsAttribute

The dictionary of font traits for stylistic information.

iOS 3.2+iPadOS 3.2+Mac Catalyst 13.1+macOS 10.5+tvOS 9.0+visionOS 1.0+watchOS 2.0+

``` source
let kCTFontTraitsAttribute: CFString
```

## Discussion

See Accessing Font Traits for the list of font traits. The value associated with this key is a CFDictionary object.
