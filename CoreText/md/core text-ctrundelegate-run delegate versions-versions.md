# Run Delegate Versions

The version of the run delegate.

## Overview

Set the version field of the CTRunDelegateCallbacks structure to `kCTRunDelegateCurrentVersion` when creating a CTRunDelegate object with a call to CTRunDelegateCreate(_:_:).

## Topics

### Constants

var kCTRunDelegateVersion1: Int

Version 1 of the run delegate.

var kCTRunDelegateCurrentVersion: Int

The current version of the run delegate.
