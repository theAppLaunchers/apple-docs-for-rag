# kCTFontManagerBundleIdentifier

macOS 10.6+

``` source
let kCTFontManagerBundleIdentifier: CFString
```

## Discussion

The Font Manager bundle identifier to be used when getting or setting global auto-activation using CTFontManagerGetAutoActivationSetting(_:) or CTFontManagerSetAutoActivationSetting(_:_:).
