# kCTFontDownloadableAttribute

The font downloadable state.

iOS 6.0+iPadOS 6.0+Mac Catalyst 13.1+macOS 10.8+tvOS 9.0+visionOS 1.0+watchOS 2.0+

``` source
let kCTFontDownloadableAttribute: CFString
```

## Discussion

The value associated with this key is a doc://com.apple.documentation/documentation/corefoundation/cfboolean-s0p. If it is kCFBooleanTrue, Core Text attempts to download a font if necessary when matching a descriptor.
