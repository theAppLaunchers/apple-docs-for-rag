# watchOS 6.2 Release Notes

Update your apps to use new features, and test your apps against API changes.

## Overview

The watchOS 6.2 SDK provides support to develop watchOS apps for Apple Watch devices running watchOS 6.2. The SDK comes bundled with Xcode 11.4, available from the Mac App Store. For information on the compatibility requirements for Xcode 11.4, see Xcode 11.4 Release Notes.

### Authentication

#### New Features

- Support for ASWebAuthenticationSession is now available when using Apple Watch Series 3 and later. Using this, apps can now authenticate users via web-based logins such as OAuth. (51821703)

### StoreKit

#### New Features

- The StoreKit framework is now available for use in watchOS apps.

### Xcode

#### Known Issues

- Simulator may take longer than expected to boot watchOS. (59961827)
