# watchOS 7.3 Release Notes

Update your apps to use new features, and test your apps against API changes.

## Overview

The watchOS 7.3 SDK provides support to develop watchOS apps for Apple Watch devices running watchOS 7.3. The SDK comes bundled with Xcode 12.4, available from the Mac App Store. For information on the compatibility requirements for Xcode 12.4, see Xcode 12.4 Release Notes.

### General

There are no new release notes for this software update.
