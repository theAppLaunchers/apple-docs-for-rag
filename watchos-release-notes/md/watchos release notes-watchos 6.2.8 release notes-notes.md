# watchOS 6.2.8 Release Notes

Update your apps to use new features, and test your apps against API changes.

## Overview

The watchOS 6.2 SDK provides support to develop watchOS apps for Apple Watch devices running watchOS 6.2.8. The SDK comes bundled with Xcode 11.5, available from the Mac App Store. For information on the compatibility requirements for Xcode 11.5, see Xcode 11.5 Release Notes.

### General

There are no release notes for this software update.
