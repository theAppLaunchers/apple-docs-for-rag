# watchOS 5.3 Release Notes

Update your apps to use new features, and test your apps against API changes.

## Overview

The watchOS 5.3 SDK provides support for developing watchOS apps for Apple Watch devices running watchOS 5.3. The SDK comes bundled with Xcode 10.3 available from the Mac App Store. For information on the compatibility requirements for Xcode 10.3, see Xcode 10.3 Release Notes.
