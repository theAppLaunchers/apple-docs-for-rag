# watchOS 10.5 Release Notes

Update your apps to use new features, and test your apps against API changes.

## Overview

The watchOS 10.5 SDK provides support to develop watchOS apps for Apple Watch devices running watchOS 10.5. The SDK comes bundled with Xcode 15.4, available from the Mac App Store. For information on the compatibility requirements for Xcode 15.4, see Xcode 15.4 Release Notes.

### StoreKit

#### Resolved Issues

- Fixed an issue where the Terms of Service button on `SubscriptionStoreView` could cause a crash when tapped. (123383479)

- Fixed: Resolved an issue causing ExternalPurchaseLink.open() to not open the external purchase URL. (125858142)
