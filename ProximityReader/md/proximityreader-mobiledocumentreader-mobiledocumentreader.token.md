# MobileDocumentReader.Token

A reader token generated by your server.

iOS 17.0+iPadOS 17.0+Mac Catalyst 17.0+

``` source
struct Token
```

## Overview

Use the `Token` to create a MobileDocumentReaderSession. You can cache the token and reuse it to create new sessions for up to 48 hours.
