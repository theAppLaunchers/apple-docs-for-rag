# init(absoluteDate:)

Creates and returns an alarm with an absolute date.

iOS 4.0+iPadOS 4.0+Mac Catalyst 13.1+macOS 10.8+visionOS 1.0+watchOS 2.0+

``` source
init(absoluteDate date: Date)
```

## Parameters 

`date`  

The date for the alarm.

## Return Value

The created alarm.
