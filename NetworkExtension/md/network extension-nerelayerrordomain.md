# NERelayErrorDomain

The domain for errors resulting from calls to the relay manager.

iOS 17.0+iPadOS 17.0+Mac Catalyst 17.0+macOS 14.0+tvOS 17.0+visionOS 1.0+

``` source
let NERelayErrorDomain: String
```

## Discussion

Match this constant to the domain of an NSError encountered when calling methods on NERelayManager. The NERelayManagerError enumeration defines possible code values for these errors.
