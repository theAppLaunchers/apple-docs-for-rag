# init(\_:)

Filters data for the provided device models.

iOS 16.0+iPadOS 16.0+Mac Catalyst 16.0+

``` source
init(_ models: Set)
```

## Discussion

Use an empty set to filter data for all devices that are sharing activity data with the current device.
