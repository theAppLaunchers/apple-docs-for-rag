# AdGroupDisplayStatus

The status of the ad group.

Search Ads 4.0+

``` source
string AdGroupDisplayStatus
```

## Possible Values 

`RUNNING`  

The AdGroup is running in the campaign.

`ON_HOLD`  

The AdGroup is on hold.

`PAUSED`  

The AdGroup is paused.

`DELETED`  

The AdGroup has been deleted.
