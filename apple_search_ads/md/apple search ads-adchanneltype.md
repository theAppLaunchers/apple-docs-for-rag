# AdChannelType

The channel type of an ad in a campaign.

Search Ads 4.0+

``` source
string AdChannelType
```

## Possible Values 

`SEARCH`  

When the SupplySource is `APPSTORE_SEARCH_RESULTS`, the AdChannelType needs to be `SEARCH`.

`DISPLAY`  

When `the` SupplySource is `APPSTORE_SEARCH_TAB, APPSTORE_PRODUCT_PAGES_BROWSE`, or `APPSTORE_TODAY_TAB`, the AdChannelType needs to be `DISPLAY`.
