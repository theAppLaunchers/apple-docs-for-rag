# KeywordInsights

The object that contains bid recommendations.

Search Ads 3.0+

``` source
object KeywordInsights
```

## Properties

`bidRecommendation`

KeywordBidRecommendation

A bid recommendation for a keyword.

## Discussion

A `bidRecommendation` helps you apply your bid strategy to exact match keywords and similar keywords in a broad match. Use `bidRecommendation` to optimize campaign performance through your Search Match discovery campaigns. For more information, see the Build a Campaign Keywords Strategy section in Ad Groups.
