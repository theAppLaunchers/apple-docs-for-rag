# init(stringValue:)

Creates a new instance from the given string.

iOS 18.0+iPadOS 18.0+Mac Catalyst 18.0+macOS 15.0+tvOS 18.0+visionOS 2.0+watchOS 11.0+Swift 5.9+

``` source
init?(stringValue: String)
```

## Parameters 

`stringValue`  

The string value of the desired key.

## Discussion

If the string passed as `stringValue` does not correspond to any instance of this type, the result is `nil`.
