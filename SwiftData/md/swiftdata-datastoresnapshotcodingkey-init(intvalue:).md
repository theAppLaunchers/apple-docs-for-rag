# init(intValue:)

Creates a new instance from the specified integer.

iOS 18.0+iPadOS 18.0+Mac Catalyst 18.0+macOS 15.0+tvOS 18.0+visionOS 2.0+watchOS 11.0+Swift 5.9+

``` source
init?(intValue: Int)
```

## Parameters 

`intValue`  

The integer value of the desired key.

## Discussion

If the value passed as `intValue` does not correspond to any instance of this type, the result is `nil`.
