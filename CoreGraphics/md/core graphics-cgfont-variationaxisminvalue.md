# variationAxisMinValue

The key used to obtain the minimum variation axis value from a variation axis dictionary.

iOS 2.0+iPadOS 2.0+Mac Catalyst 13.1+macOS 10.4+tvOSvisionOS 1.0+watchOS 2.0+

``` source
class let variationAxisMinValue: CFString
```

## Description

The value obtained with this key is a CFNumber that specifies the minimum value of the variation axis.
