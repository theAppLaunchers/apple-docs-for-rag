# variationAxisName

The key used to obtain the variation axis name from a variation axis dictionary.

iOS 2.0+iPadOS 2.0+Mac Catalyst 13.1+macOS 10.4+tvOSvisionOS 1.0+watchOS 2.0+

``` source
class let variationAxisName: CFString
```

## Description

The value obtained with this key is a CFString that specifies the name of the variation axis.
