# kAudioChannelLayoutTag_DVD_20

A DVD 5.1-channel audio layout.

iOS 2.0+iPadOS 2.0+macOS 10.2+tvOS 9.0+visionOS 1.0+watchOS 3.0+

``` source
var kAudioChannelLayoutTag_DVD_20: AudioChannelLayoutTag { get }
```

## Discussion

Your channels must be laid out in the following order:

- Left

- Right

- Left surround

- Right surround

- Center

- Low-fequency effects.
