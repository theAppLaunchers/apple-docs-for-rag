# kAudioChannelLayoutTag_MPEG_7_1_B

An MPEG 7.1-channel, configuration B, audio layout.

iOS 2.0+iPadOS 2.0+macOS 10.2+tvOS 9.0+visionOS 1.0+watchOS 3.0+

``` source
var kAudioChannelLayoutTag_MPEG_7_1_B: AudioChannelLayoutTag { get }
```

## Discussion

See ISO/IEC 13818-7 MPEG2-AAC, Table 3.1 for more information. Your channels must be laid out in the following order:

- Center

- Left center

- Right center

- Left

- Right

- Left surround

- Right surround

- Low-frequency effects
