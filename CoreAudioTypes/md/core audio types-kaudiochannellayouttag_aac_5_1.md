# kAudioChannelLayoutTag_AAC_5_1

An AAC 5.1-channel surround-based layout.

iOS 2.0+iPadOS 2.0+macOS 10.3+tvOS 9.0+visionOS 1.0+watchOS 3.0+

``` source
var kAudioChannelLayoutTag_AAC_5_1: AudioChannelLayoutTag { get }
```

## Discussion

Your channels must be laid out in the following order:

- Center

- Left

- Right

- Left surround

- Right surround

- Low-frequency effects
