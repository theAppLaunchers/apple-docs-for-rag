# kAudioChannelLayoutTag_ITU_3_2

An ITU 3.2-channel audio layout.

iOS 2.0+iPadOS 2.0+macOS 10.2+tvOS 9.0+visionOS 1.0+watchOS 3.0+

``` source
var kAudioChannelLayoutTag_ITU_3_2: AudioChannelLayoutTag { get }
```

## Discussion

Your channels must be laid out in the following order:

- Left

- Right

- Center

- Left surround

- Right surround
