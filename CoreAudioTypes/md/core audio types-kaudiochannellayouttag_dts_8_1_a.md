# kAudioChannelLayoutTag_DTS_8_1_A

A Blu-ray Disc audio layout, defined by DTS (Digital Theater Systems Ltd.).

iOS 4.0+iPadOS 4.0+macOS 10.7+tvOS 9.0+visionOS 1.0+watchOS 3.0+

``` source
var kAudioChannelLayoutTag_DTS_8_1_A: AudioChannelLayoutTag { get }
```

## Discussion

Your channel layout must be in the following order:

- Left center

- Right center

- Left

- Right

- Left surround

- Right surround

- Left rear surround

- right rear surround

- Low-frequency effects
