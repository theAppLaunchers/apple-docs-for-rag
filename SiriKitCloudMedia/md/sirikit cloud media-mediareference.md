# MediaReference

A way of identifying the current media item rather than with metadata.

SiriKit Cloud Media 1.0.2+

``` source
string MediaReference
```

## Possible Values 

`currentlyPlaying`  

The media item the device is playing.

`unknown`  

An unspecified reference.

`my`  

Information specific to the user, as in *Play my Top 10 playlist*.
