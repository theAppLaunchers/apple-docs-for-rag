# init(yaw:pitch:roll:)

Creates the limit object.

iOS 17.0+iPadOS 17.0+Mac Catalyst 17.0+macOS 14.0+

``` source
init(
    yaw: DockAccessory.Limits.Limit?,
    pitch: DockAccessory.Limits.Limit?,
    roll: DockAccessory.Limits.Limit?
)
```

## Parameters 

`yaw`  

The left and right limit.

`pitch`  

The up and down limit.

`roll`  

The side to side limit.
