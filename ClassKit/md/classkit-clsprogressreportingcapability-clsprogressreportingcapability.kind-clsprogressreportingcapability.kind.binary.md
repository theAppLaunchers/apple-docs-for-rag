# CLSProgressReportingCapability.Kind.binary

A binary outcome for the task, like true or false.

iOS 14.0+iPadOS 14.0+Mac Catalyst 14.0+macOS 11.0+visionOS 1.0+

``` source
case binary
```

## Discussion

Add this capability to indicate that a context reports a binary condition at the end of a task, like pass or fail, using a CLSBinaryItem instance.
