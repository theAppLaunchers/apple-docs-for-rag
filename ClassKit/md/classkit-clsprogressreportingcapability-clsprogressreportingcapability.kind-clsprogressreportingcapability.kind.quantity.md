# CLSProgressReportingCapability.Kind.quantity

A discrete value.

iOS 14.0+iPadOS 14.0+Mac Catalyst 14.0+macOS 11.0+visionOS 1.0+

``` source
case quantity
```

## Discussion

Add this capability to indicate that a context reports an arbitrary numerical result at the end of a task using a CLSQuantityItem instance.
