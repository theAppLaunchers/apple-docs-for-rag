Instance Property

# allowsMultipleSelection

A Boolean value indicating whether the user is allowed to select multiple identities.

macOS 10.5+

``` source
var allowsMultipleSelection: Bool { get set }
```

## Discussion

The value of this property is true if the user can select multiple records; otherwise, false. The default value is false.
