# init(rawValue:)

Initializes an event mask value.

Mac Catalyst 13.0+macOS 10.0+

``` source
init(rawValue: UInt32)
```

## Parameters 

`rawValue`  

The bitwise `OR` of one or more of the event mask constants used to initialize an event mask
