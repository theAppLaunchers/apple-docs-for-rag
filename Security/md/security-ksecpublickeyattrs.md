# kSecPublicKeyAttrs

A key whose value is a dictionary of cryptographic key attributes specific to a public key.

iOS 2.0+iPadOS 2.0+Mac Catalyst 13.1+macOS 10.8+tvOS 4.0+visionOS 1.0+watchOS 1.0+

``` source
let kSecPublicKeyAttrs: CFString
```

## Mentioned in 

Generating New Cryptographic Keys

## Discussion

The corresponding value is a CFDictionary containing key-value pairs for attributes specific to the public key to be generated.
