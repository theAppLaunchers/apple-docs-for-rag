# SecItemAttr.protocolItemAttr

Identifies the protocol attribute.

Mac Catalyst 13.0+macOS 10.0+

``` source
case protocolItemAttr
```

## Discussion

You use this tag to set or get a value of type `SecProtocolType` that represents the Internet protocol. For possible protocol type values, see SecProtocolType. This is unique to AppleShare and Internet password attributes.
