# Code Signing Architecture Flags

Use these supplemental flags to get static code.

## Overview

These flags supplement the flags described in SecCSFlags. Use these additional constants with the `flags` parameter of the SecCodeCopyStaticCode(_:_:_:) function.

## Topics

### Constants

var kSecCSUseAllArchitectures: UInt32

Flag for requesting all architectures.
