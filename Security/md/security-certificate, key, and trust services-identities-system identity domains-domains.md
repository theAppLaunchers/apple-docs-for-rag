# System Identity Domains

Set or obtain a system identity for domains.

## Overview

These constants can be used with the SecIdentitySetSystemIdentity(_:_:) and SecIdentityCopySystemIdentity(_:_:_:) functions.

## Topics

### Constants

let kSecIdentityDomainDefault: CFString

The system-wide default identity.

let kSecIdentityDomainKerberosKDC: CFString

Kerberos Key Distribution Center (KDC) identity.
