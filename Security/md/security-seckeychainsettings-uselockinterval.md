# useLockInterval

A Boolean value indicating whether the keychain automatically locks after a certain period of time.

Mac Catalyst 13.0+macOS 10.0+

``` source
var useLockInterval: DarwinBoolean
```

## Discussion

Use lockInterval to indicate the time in seconds after which the keychain should automatically be locked.
