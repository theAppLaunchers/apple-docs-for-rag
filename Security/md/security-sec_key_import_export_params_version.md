# SEC_KEY_IMPORT_EXPORT_PARAMS_VERSION

The import/export parameter structure version.

macOS 10.0+

``` source
var SEC_KEY_IMPORT_EXPORT_PARAMS_VERSION: Int32 { get }
```

## Discussion

Use this constant to set the version property of an instance of the SecItemImportExportKeyParameters structure.
