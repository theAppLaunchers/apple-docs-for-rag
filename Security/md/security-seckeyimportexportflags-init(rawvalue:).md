# init(rawValue:)

Initialize a key import/export flag structure.

macOS 10.0+

``` source
init(rawValue: UInt32)
```

## Parameters 

`rawValue`  

An initial value for the structure composed as the bitwise `OR` of zero or more of the valid values.
