# HKMetadataKeyBarometricPressure

The metadata key for the barometric pressure associated with a sample.

iOS 14.0+iPadOS 14.0+Mac Catalyst 14.0+macOS 13.0+visionOS 1.0+watchOS 7.0+

``` source
let HKMetadataKeyBarometricPressure: String
```

## Discussion

This key takes an HKQuantity value that measures the barometric pressure in units of pressure, such as atmosphere(), pascal(), or millimeterOfMercury().
