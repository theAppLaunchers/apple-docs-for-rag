# init(\_:)

Creates a document type using the provided identifier.

iOS 15.0+iPadOS 15.0+Mac Catalyst 15.0+macOS 13.0+visionOSwatchOS 8.0+

``` source
convenience init(_ identifier: HKDocumentTypeIdentifier)
```

## Parameters 

`identifier`  

An identifier that specifies the type of document to store in the sample. For a complete list of document type identifiers, see HKDocumentTypeIdentifier.
