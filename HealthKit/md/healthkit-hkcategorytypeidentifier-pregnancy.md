# pregnancy

A category type that records pregnancy.

iOS 14.3+iPadOS 14.3+Mac Catalyst 14.3+macOS 13.0+visionOS 1.0+watchOS 7.2+

``` source
static let pregnancy: HKCategoryTypeIdentifier
```

## Discussion

Use a HKCategoryValue.notApplicable value with samples of this type.
