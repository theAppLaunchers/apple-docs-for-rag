# HKMetadataKeyIndoorBikeDistance

The workout distance displayed by a connected GymKit exercise bike.

iOS 12.0+iPadOS 12.0+Mac Catalyst 13.0+macOS 13.0+visionOS 1.0+watchOS 5.0+

``` source
let HKMetadataKeyIndoorBikeDistance: String
```

## Discussion

Set this key on a workout sample representing exercise on a GymKit exercise bike. Set its value to an HKQuantity object with a length unit.
