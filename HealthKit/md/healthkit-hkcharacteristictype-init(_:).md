# init(\_:)

Creates a characteristic type using the provided identifier.

iOS 15.0+iPadOS 15.0+Mac Catalyst 15.0+macOS 13.0+visionOSwatchOS 8.0+

``` source
convenience init(_ identifier: HKCharacteristicTypeIdentifier)
```

## Parameters 

`identifier`  

An identifier that specifies the type of characteristic data. For a complete list of characteristic type identifiers, see HKCharacteristicTypeIdentifier.
