# HKWorkoutActivityType.transition

A constant for the transition time between activities in a multisport workout.

iOS 16.0+iPadOS 16.0+Mac Catalyst 16.0+macOS 13.0+visionOS 1.0+watchOS 9.0+

case transition

## Mentioned in 

Dividing a HealthKit workout into activities

## Discussion

Use this constant to create an HKWorkoutActivity instance that represents the transition between sports in a HKWorkoutActivityType.swimBikeRun workout.
