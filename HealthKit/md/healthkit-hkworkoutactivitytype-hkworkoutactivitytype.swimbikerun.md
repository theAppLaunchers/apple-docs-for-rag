# HKWorkoutActivityType.swimBikeRun

The constant for multisport activities like triathlons.

iOS 16.0+iPadOS 16.0+Mac Catalyst 16.0+macOS 13.0+visionOS 1.0+watchOS 9.0+

``` source
case swimBikeRun
```

## Mentioned in 

Dividing a HealthKit workout into activities

## Discussion

Use this constant to create workouts that contain the HKWorkoutActivityType.swimming, HKWorkoutActivityType.cycling, and HKWorkoutActivityType.running activities. For more information, see HKWorkoutActivity.
