# HKMetadataKeyCrossTrainerDistance

The workout distance displayed by a connected GymKit cross-trainer machine.

iOS 12.0+iPadOS 12.0+Mac Catalyst 13.0+macOS 13.0+visionOS 1.0+watchOS 5.0+

``` source
let HKMetadataKeyCrossTrainerDistance: String
```

## Discussion

Set this key on a workout sample representing exercise on a GymKit cross-trainer machine (such as an elliptical cross-trainer). Set its value to an HKQuantity object with a length unit.
