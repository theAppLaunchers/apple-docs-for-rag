# init(predicate:)

Instantiates an activity summary query descriptor.

iOS 15.4+iPadOS 15.4+Mac Catalyst 15.4+macOS 13.0+visionOSwatchOS 8.5+

``` source
init(predicate: NSPredicate?)
```

## Parameters 

`predicate`  

A predicate that limits the results that the query returns. For a list of convenience methods for building predicates, see HKQuery.
