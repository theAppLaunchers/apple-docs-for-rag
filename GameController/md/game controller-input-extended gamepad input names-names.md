# Extended gamepad input names

Constants for names of extended gamepad elements.

## Overview

Use these constants as the keys when accessing GCExtendedGamepad elements by name using the allElements property, `subscript(_:)` notation, or similar GCPhysicalInputProfile APIs.

## Topics

### Shoulder button names

var GCInputLeftShoulder: String

The name of the left shoulder button.

var GCInputRightShoulder: String

The name of the right shoulder button.

### Trigger names

var GCInputLeftTrigger: String

The name of the left trigger.

var GCInputRightTrigger: String

The name of the right trigger.

### Face button names
