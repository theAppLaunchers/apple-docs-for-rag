# GCDeviceCursor

A control element for the cursor used as a directional pad.

iOS 14.0+iPadOS 14.0+Mac Catalyst 14.0+macOS 11.0+tvOS 14.0+visionOS 1.0+

``` source
class GCDeviceCursor
```

## Overview

This controller element allows you to use the cursor as a directional pad with the values of the xAxis and yAxis elements scaled to the width and height of the screen, not ranging from `-1` to `1`.
