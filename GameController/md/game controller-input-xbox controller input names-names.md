# Xbox controller input names

Constants for names of Xbox elements.

## Overview

Use these constants as the keys when accessing GCXboxGamepad elements by name using the allElements property, `subscript(_:)` notation, or similar GCPhysicalInputProfile APIs.

## Topics

### Xbox controller button names

var GCInputButtonShare: String

The name of the share button.

var GCInputXboxPaddleOne: String

The name for the controller’s P1 paddle button.

var GCInputXboxPaddleTwo: String

The name for the controller’s P2 paddle button.

var GCInputXboxPaddleThree: String

The name for the controller’s P3 paddle button.

var GCInputXboxPaddleFour: String

The name for the controller’s P4 paddle button.
