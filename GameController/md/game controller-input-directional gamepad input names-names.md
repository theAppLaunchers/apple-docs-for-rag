# Directional Gamepad Input Names

Constants for names of directional pad elements.

## Topics

### Directional Pad Names

let GCInputDirectionalDpad: String

The name of the controller’s primary directional pad element.

let GCInputDirectionalCardinalDpad: String

The name of the controller’s optional secondary directional pad element.

### Directional Pad Button Names

let GCInputDirectionalTouchSurfaceButton: String

The name of the controller’s touch surface button.

let GCInputDirectionalCenterButton: String

The name of the controller’s optional button on the directional gamepad.
