# reversesTitleShadowWhenHighlighted

A Boolean value that determines whether the title shadow changes when the button is highlighted.

iOS 2.0–15.0DeprecatediPadOS 2.0–15.0DeprecatedMac Catalyst 13.1–15.0DeprecatedtvOS 2.0–15.0DeprecatedvisionOS 1.0–1.0Deprecated

``` source
@MainActor
var reversesTitleShadowWhenHighlighted: Bool { get set }
```

## Discussion

If true, the shadow changes from engrave to emboss appearance when highlighted. The default value is false.
