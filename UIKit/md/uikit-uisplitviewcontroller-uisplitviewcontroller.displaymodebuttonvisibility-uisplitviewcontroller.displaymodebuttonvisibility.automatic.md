# UISplitViewController.DisplayModeButtonVisibility.automatic

A constant that automatically determines the visibility of the display mode button.

iOS 14.5+iPadOS 14.5+Mac Catalyst 14.5+tvOS 14.5+visionOS 1.0+

``` source
case automatic
```

## Discussion

With this behavior, setting presentsWithGesture to false hides the display mode button. This behavior is the default.
