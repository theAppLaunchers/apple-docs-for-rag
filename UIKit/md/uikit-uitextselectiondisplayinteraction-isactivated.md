# isActivated

A Boolean value that indicates whether to display the system selection UI.

iOS 17.0+iPadOS 17.0+Mac Catalyst 17.0+tvOS 17.0+visionOS 1.0+

``` source
@MainActor
var isActivated: Bool { get set }
```

## Discussion

Set this property to true when you want the system to display the selection UI. Set the property to false to hide the selection UI again.
