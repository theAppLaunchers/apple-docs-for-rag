# isVisible

A Boolean value that indicates whether the receiver is displayed.

iOS 2.0–9.0DeprecatediPadOS 2.0–9.0DeprecatedMac Catalyst 13.1–13.1Deprecated


## Discussion

If true, the receiver is displayed; otherwise, false.
