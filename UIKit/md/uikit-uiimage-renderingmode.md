# renderingMode

A setting that determines how the app renders an image.

iOS 7.0+iPadOS 7.0+Mac Catalyst 13.1+tvOSvisionOS 1.0+watchOS 2.0+

``` source
var renderingMode: UIImage.RenderingMode { get }
```

## Discussion

The default rendering mode is UIImage.RenderingMode.automatic.
