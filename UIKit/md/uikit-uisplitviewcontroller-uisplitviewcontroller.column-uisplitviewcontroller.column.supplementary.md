# UISplitViewController.Column.supplementary

The column for the supplementary view controller.

iOS 14.0+iPadOS 14.0+Mac Catalyst 14.0+tvOS 14.0+visionOS 1.0+

``` source
case supplementary
```

## Discussion

This value only takes effect when the split view controller’s style property is UISplitViewController.Style.tripleColumn.
