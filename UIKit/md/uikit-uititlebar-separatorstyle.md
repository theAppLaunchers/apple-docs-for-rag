# separatorStyle

The type of separator that the app displays between the title bar and content of a window.

Mac Catalyst 14.0+

``` source
@MainActor
var separatorStyle: UITitlebarSeparatorStyle { get set }
```

## Discussion

The default value is UITitlebarSeparatorStyle.automatic.
