# capHeight

The font’s cap height information.

iOS 2.0+iPadOS 2.0+Mac Catalyst 13.1+tvOSvisionOS 1.0+watchOS 2.0+

``` source
var capHeight: CGFloat { get }
```

## Discussion

This value measures (in points) the height of a capital character.
