# UISplitViewController.DisplayModeButtonVisibility.always

A constant that allows the display mode button to always appear.

iOS 14.5+iPadOS 14.5+Mac Catalyst 14.5+tvOS 14.5+visionOS 1.0+

``` source
case always
```

## Discussion

With this behavior, the display mode button can appear even if presentsWithGesture is false.
