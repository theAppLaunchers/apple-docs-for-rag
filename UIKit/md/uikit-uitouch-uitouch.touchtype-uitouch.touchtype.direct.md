# UITouch.TouchType.direct

A touch resulting from direct contact with the screen.

iOS 9.0+iPadOS 9.0+Mac Catalyst 13.1+tvOS 9.0+visionOS 1.0+

``` source
case direct
```

## Discussion

Direct touches occur when the user’s finger touches the screen.
