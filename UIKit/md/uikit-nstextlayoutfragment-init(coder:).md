# init(coder:)

Creates a new layout fragment with the coder you provide.

iOS 15.0+iPadOS 15.0+Mac Catalyst 15.0+tvOS 15.0+visionOS 1.0+

``` source
init?(coder: NSCoder)
```

## Parameters 

`coder`  

A coder that conforms to NSCoder.
