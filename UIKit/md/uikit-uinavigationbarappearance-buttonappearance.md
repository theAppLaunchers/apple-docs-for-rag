# buttonAppearance

The appearance attributes for plain bar button items in the navigation bar.

iOS 13.0+iPadOS 13.0+Mac Catalyst 13.1+tvOS 13.0+visionOS 1.0+

``` source
@NSCopying @MainActor
var buttonAppearance: UIBarButtonItemAppearance { get set }
```

## Discussion

To configure the appearance of the back or done buttons separately, use the backButtonAppearance and doneButtonAppearance properties instead.
