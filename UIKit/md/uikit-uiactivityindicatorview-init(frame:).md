# init(frame:)

Creates an activity indicator with the specified frame rectangle.

iOS 2.0+iPadOS 2.0+Mac Catalyst 13.1+tvOSvisionOS 1.0+

``` source
@MainActor
init(frame: CGRect)
```

## Parameters 

`frame`  

The frame rectangle for the view, specified in points.
