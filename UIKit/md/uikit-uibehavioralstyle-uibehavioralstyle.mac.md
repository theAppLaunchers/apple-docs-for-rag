# UIBehavioralStyle.mac

A style that indicates that a control appears and behaves as it does in macOS.

iOS 15.0+iPadOS 15.0+Mac Catalyst 15.0+visionOS 1.0+

case mac

## Mentioned in 

Building a desktop-class iPad app

Choosing a user interface idiom for your Mac app

## Discussion

This style effects a control’s behavior only when the app’s user interface idiom is UIUserInterfaceIdiom.mac.
