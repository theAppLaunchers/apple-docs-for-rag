# init(url:)

Creates a printer destination with the specified address.

iOS 14.5+iPadOS 14.5+Mac Catalyst 14.5+visionOS 1.0+

``` source
@MainActor
init(url: URL)
```

## Parameters 

`url`  

A URL that identifies the location of the printer.

## Return Value

An object representing the printer.
