# init(coder:)

Initializes and returns a newly allocated text list item.

iOS 7.0+iPadOS 7.0+Mac Catalyst 13.1+tvOS 9.0+visionOS 1.0+watchOS 2.0+

``` source
init?(coder: NSCoder)
```

## Parameters 

`coder`  

An instance of NSCoder.
