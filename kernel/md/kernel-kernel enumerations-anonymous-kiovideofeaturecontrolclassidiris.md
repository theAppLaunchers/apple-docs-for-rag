# kIOVideoFeatureControlClassIDIris

macOS 10.12+

``` source
kIOVideoFeatureControlClassIDIris = 'iris'
```

## Discussion

A IOVideoFeatureControl that controls a mechanical lens iris. The units for the control's absolute value are an F number (F).
