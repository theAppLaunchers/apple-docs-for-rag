# getBusNumber

Accessor to return the PCI device's assigned bus number.

``` source
virtual UInt8 getBusNumber(
 void ); 
```

## Return Value

The 8-bit value of device's PCI bus number.

## Overview

This method is an accessor to return the PCI device's assigned bus number.
