# kIOPMPowerOn

macOS 10.12+

``` source
kIOPMPowerOn = 0x00000002
```

## Discussion

Indicates the device is on, requires power, and provides power. Useful as a: Capability, InputPowerRequirement, OutputPowerCharacter
