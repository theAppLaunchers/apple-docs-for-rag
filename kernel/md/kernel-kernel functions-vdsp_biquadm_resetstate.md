# vDSP_biquadm_ResetState

Resets the filter state of a single-precision multichannel biquad IIR filter object.

macOS 10.9+

``` source
void vDSP_biquadm_ResetState(vDSP_biquadm_Setup __setup);
```

## Parameters 

`__setup`
