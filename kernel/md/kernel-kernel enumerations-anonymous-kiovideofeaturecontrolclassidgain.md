# kIOVideoFeatureControlClassIDGain

macOS 10.12+

``` source
kIOVideoFeatureControlClassIDGain = 'gain'
```

## Discussion

A IOVideoFeatureControl that controls the amplification of the signal. The units for the control's absolute value are decibels (dB).
