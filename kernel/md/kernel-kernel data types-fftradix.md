# FFTRadix

The size of the FFT decomposition.

macOS 10.13+

``` source
typedef int FFTRadix;
```

## Discussion

An `FFTRadix` value is passed as an argument to vDSP_create_fftsetup or vDSP_create_fftsetupD.
