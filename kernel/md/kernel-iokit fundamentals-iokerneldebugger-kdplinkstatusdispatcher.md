# kdpLinkStatusDispatcher

The KDP link status dispatch function.

``` source
static UInt32 kdpLinkStatusDispatcher(
 void); 
```

## Return Value

Return link status.

## Overview

Field KDP link status requests, then dispatches the call to the registered link up handler.
