# kIOMediaAttributeEjectableMask

macOS 10.12+

``` source
kIOMediaAttributeEjectableMask = 0x00000001
```

## Discussion

Indicates whether the media is ejectable from the drive mechanism under software control. Implies kIOMediaAttributeRemovableMask.
