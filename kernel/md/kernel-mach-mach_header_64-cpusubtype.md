# cpusubtype

An integer specifying the exact model of the CPU. To run on all PowerPC processors supported by the macOS kernel, this should be set to `CPU_SUBTYPE_POWERPC_ALL`.

macOS 10.6+

``` source
cpu_subtype_t cpusubtype;
```
