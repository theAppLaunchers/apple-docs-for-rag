# kIOHIDOptionsTypeSeizeDevice

macOS 10.12+

``` source
kIOHIDOptionsTypeSeizeDevice = 0x01
```

## Discussion

Used to open exclusive communication with the device. This will prevent the system and other clients from receiving events from the device.
