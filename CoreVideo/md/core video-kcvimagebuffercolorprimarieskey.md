# kCVImageBufferColorPrimariesKey

A key to the color primaries gamut for the image buffer.

iOS 4.0+iPadOS 4.0+Mac Catalyst 13.1+macOS 10.5+tvOS 9.0+visionOS 1.0+watchOS 2.0+

``` source
let kCVImageBufferColorPrimariesKey: CFString
```

## Discussion

The color primaries gamut value is of type CFString. See Image Buffer Color Primaries Constants for more information.
