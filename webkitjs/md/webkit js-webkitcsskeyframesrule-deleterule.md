# deleteRule

Removes a keyframe rule from the collection of keyframes.

``` source
void deleteRule (in DOMString key);
```

## Parameters 

`key`  

A selector for the rule that is either a percentage or the keyword `from` or `to`.

## See Also

### Changing Rules

insertRule

findRule
