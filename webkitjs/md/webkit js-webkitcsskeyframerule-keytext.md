# keyText

A keyframe selector.

``` source
attribute DOMString keyText
```

## Overview

A keyframe selector represented as a percentage value or the keyword `from` or `to`. The selector is used to specify the percentage along the duration of the animation. The keyframe is specified by the block of property values declared for the selector. The keyword `from` is equivalent to `0` percent and `to` is equivalent to `100` percent.
