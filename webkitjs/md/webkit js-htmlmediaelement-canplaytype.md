# canPlayType

Returns whether the media element supports the specified MIME type.

Safari Desktop 4.0+Safari Mobile 3.0+

``` source
DOMString canPlayType(
    DOMString type
);
```

## Return Value

The possible string values are: “no”, “probably” and “maybe”.
