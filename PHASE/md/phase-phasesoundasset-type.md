# type

The type of sound asset.

iOS 15.0+iPadOS 15.0+Mac Catalyst 15.0+macOS 12.0+tvOS 17.0+visionOS 1.0+

``` source
var type: PHASEAsset.AssetType { get }
```

## Discussion

The framework sets the value of this property to the type you pass into registerSoundAsset(url:identifier:assetType:channelLayout:normalizationMode:).
