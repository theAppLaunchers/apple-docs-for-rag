# activate(timeToTargetOverride:)

Applies settings with an overriden fade duration.

iOS 15.0+iPadOS 15.0+Mac Catalyst 15.0+macOS 12.0+tvOS 17.0+visionOS 1.0+

``` source
func activate(timeToTargetOverride: Double)
```

## Parameters 

`timeToTargetOverride`  

A duration in which the engine fades the settings from their original value to their new value. Overrides timeToTarget.
