# PHASECurveType.inverseSine

An inverse sine curve.

iOS 15.0+iPadOS 15.0+Mac Catalyst 15.0+macOS 12.0+tvOS 15.0+visionOS 1.0+

``` source
case inverseSine
```

## Discussion

The function `y` `=` `sin^-1(x)` shapes this curve. The inverse sine curve behaves like PHASECurveType.sine, only rotated `90` degrees.
