# PHASECurveType.sigmoid

A sigmoid curve.

iOS 15.0+iPadOS 15.0+Mac Catalyst 15.0+macOS 12.0+tvOS 15.0+visionOS 1.0+

``` source
case sigmoid
```

## Discussion

Also known as an *s-curve*, the sigmoid curve’s path movement is slow at either end and quick in the middle.
