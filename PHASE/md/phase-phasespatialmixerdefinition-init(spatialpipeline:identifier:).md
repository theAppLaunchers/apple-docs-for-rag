# init(spatialPipeline:identifier:)

Creates a named mixer with the designated spatial pipeline.

iOS 15.0+iPadOS 15.0+Mac Catalyst 15.0+macOS 12.0+tvOS 17.0+visionOS 1.0+

``` source
convenience init(
    spatialPipeline: PHASESpatialPipeline,
    identifier: String
)
```

## Parameters 

`spatialPipeline`  

An object that features optional sound resonance effects.

`identifier`  

A unique name for the spatial mixer.
