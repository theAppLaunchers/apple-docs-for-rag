# Platform

Strings that represent Apple operating systems.

App Store Connect API 1.0+

``` source
string Platform
```

## Possible Values 

`IOS`  

`MAC_OS`  

`TV_OS`  

`VISION_OS`  

## Possible values

IOS  
A string that represents iOS.

MAC_OS  
A string that represents macOS.

TV_OS  
A string that represents tvOS.

VISION_OS  
A string that represents visionOS.
