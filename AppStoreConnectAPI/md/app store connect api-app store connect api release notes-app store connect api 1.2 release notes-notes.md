# App Store Connect API 1.2 release notes

Update your server-side code to use new features, and test your code against API changes.

## Overview

App Store Connect API version 1.2 provides resources that enable you to automate actions you take in App Store Connect.

### New features

- The new Apps resource and the App Metadata collection of resources enable automating the creation of new app versions and managing app store information.

- The new App Store Version Phased Releases resource enables automating the submission of new versions of your app to the App Store.

- The Power and Performance Metrics and Logs now enables downloading of aggregate metrics and diagnostics for App Store versions of your app.

- The Certificates resource now supports unified software signing and multiplatform app IDs, and the CapabilityType has additional bundle ID capabilities.
