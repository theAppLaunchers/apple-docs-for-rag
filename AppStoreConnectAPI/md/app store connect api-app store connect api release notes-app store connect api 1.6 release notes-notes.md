# App Store Connect API 1.6 release notes

Update your server-side code to use new features, and test your code against API changes.

## Overview

App Store Connect API version 1.6 provides resources that enable you to automate actions you take in App Store Connect.

### New features

- The API adds App Clips and App Clip Experiences.

- For TestFlight, a new attribute in BetaGroupUpdateRequest.Data.Attributes helps developers manage their iOS apps on Apple silicon Mac computers.

- The Build.Relationships.BuildBundles and BuildBundle.Relationships.BuildBundleFileSizes relationships provide developers more insight into their app binaries.

- The Apps resource adds the ability to configure a subscription status URL for production and sandbox environments in App.Attributes and AppUpdateRequest.Data.Attributes.
