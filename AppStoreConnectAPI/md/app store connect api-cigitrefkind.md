# CiGitRefKind

A string that represents the kind of a Git References resource.

App Store Connect API 1.5+

``` source
string CiGitRefKind
```

## Possible Values 

`BRANCH`  

`TAG`  

## Possible values

`BRANCH`  
The Git reference represents a branch.

`TAG`  
The Git reference represents a tag.
