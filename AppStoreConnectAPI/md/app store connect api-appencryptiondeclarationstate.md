# AppEncryptionDeclarationState

Strings that represent the review or acceptance status of an app encryption declaration submitted to Apple.

App Store Connect API 1.0+

``` source
string AppEncryptionDeclarationState
```

## Possible Values 

`CREATED`  

`IN_REVIEW`  

`APPROVED`  

`REJECTED`  

`INVALID`  

`EXPIRED`
