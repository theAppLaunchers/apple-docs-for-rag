# App Store Connect API 2.2 release notes

Update your server-side code to use new features, and test your code against API changes.

## Overview

App Store Connect API version 2.2 provides resources that enable you to automate actions you take in App Store Connect.

### New features

- Added support for getting information about and editing Sandbox Testers and clearing Sandbox Testers purchase history.

- Expanded support around App Encryption Declarations, state and documents.

- Billing Grace Periods now has additional parameters including 3 distinct durations.

### Deprecations

- The List All In-App Purchases for an App V1 endpoint is now deprecated and replaced with the List All In-App Purchases for an App endpoint.
