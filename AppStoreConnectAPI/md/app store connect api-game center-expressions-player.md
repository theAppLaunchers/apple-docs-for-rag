# Player

An object that represents a player associated with a match request.

## Overview

Use a `Player` object in the expression of a matchmaking rule to access information about a player.

### Properties

`properties`  
The game-specific properties of the game or player that initiates a match request.

`playerId`  
A unique identifier for the player.

`requestName`  
An identifier for the associated match request.
