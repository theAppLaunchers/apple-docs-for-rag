# App Store Connect API 1.1 release notes

Update your server-side code to use new features, and test your code against API changes.

## Overview

App Store Connect API version 1.1 provides resources that enable you to automate actions you take in App Store Connect.

### New features

- The API now supports provisioning, including managing bundle IDs with the Bundle IDs and Bundle ID Capabilities resources, managing certificates with the Certificates resource, registering test devices with the Devices resource, and managing provisioning profiles with Profiles.
