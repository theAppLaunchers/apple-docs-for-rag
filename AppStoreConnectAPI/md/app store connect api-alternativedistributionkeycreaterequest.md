# AlternativeDistributionKeyCreateRequest

The request body you use to create an alternative distribution key.

App Store Connect API 3.3+

``` source
object AlternativeDistributionKeyCreateRequest
```

## Properties

AlternativeDistributionKeyCreateRequest.Data

## Discussion

Use this object to create a new alternative distribution key association in App Store Connect. For more infomation about the request that includes this request body, see Add an alternative distribution key.
