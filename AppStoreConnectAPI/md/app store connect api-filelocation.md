# FileLocation

The data structure that represents a File Locations resource.

App Store Connect API 1.5+

``` source
object FileLocation
```

## Properties

`lineNumber`

`integer`

The line number of a file that contains code.

`path`

`string`

The path to the file that caused an issue.
