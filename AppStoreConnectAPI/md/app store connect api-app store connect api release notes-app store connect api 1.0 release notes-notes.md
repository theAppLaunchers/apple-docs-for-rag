# App Store Connect API 1.0 release notes

The initial version of the API.

## Overview

This REST API provides resources that enable you to automate actions you take in App Store Connect.

### New features

- Manage beta builds of your app, beta testers, and groups for Testflight with resources in Prerelease Versions and Beta Testers.

- Manage your users and their access with the Users resources. Invite users to join your team with the User Invitations resource.

- Download sales and financial reports in Sales and Finance.
