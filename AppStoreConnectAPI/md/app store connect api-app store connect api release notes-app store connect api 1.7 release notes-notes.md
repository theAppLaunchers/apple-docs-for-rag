# App Store Connect API 1.7 release notes

Update your server-side code to use new features, and test your code against API changes.

## Overview

App Store Connect API version 1.7 provides resources that enable you to automate actions you take in App Store Connect.

### New features

- The new resources in App Events and Metadata support scheduling and managing in-app events, including App Events, App Event Video Clips, App Event Screenshots, and App Event Localizations.

- New resources support creating and managing your appâ€™s custom product pages, including App Custom Product Pages, App Custom Product Page Versions, and App Custom Product Page Localizations.

- The App Store version experiments resource now supports creating and managing product page optimization treatments.

- The Review submissions and Review submission items resources support the updated App Store submission process, which lets you submit multiple items, submit without a new app version, and retrieve the submission statuses.

### Deprecations

- The App Store version submissions resource and all its related endpoints and objects are now deprecated, including: Create an App Store version submission, Delete an App Store Version Submission, AppStoreVersionSubmission, AppStoreVersionSubmissionCreateRequest, and AppStoreVersionSubmissionResponse.
