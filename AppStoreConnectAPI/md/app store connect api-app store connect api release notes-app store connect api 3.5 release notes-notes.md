# App Store Connect API 3.5 release notes

Update your server-side code to use new features, and test your code against API changes.

## Overview

App Store Connect API version 3.5 provides resources that enable you to automate actions you take in App Store Connect.

### New Features

- Developers can optionally provide a single deep-link URL for a custom project page by adding a `deepLink` attribute to AppCustomProductPageVersionCreateRequest.Data.Attributes. For more information, see Configure multiple product page versions.

- `ELAPSED_TIME_CENTISECOND` is now available in GameCenterLeaderboardFormatter.

- Developers can now more easily debug slow-launch issues for their app with the new `LAUNCHES diagnosticType` for List All Diagnostic Signatures for a Build.

- Trending insights for disk writes signatures are now available and show the impact of signature trends across app versions. List All Diagnostic Signatures for a Build now includes the `DiagnosticInsightDirection`, `DiagnosticInsightType`, and an impact breakdown by app version.

- A new attribute `BetaTesterState` is available for BetaTester.Attributes. For more information, see View and manage tester information.

### Removals

- The `IMAGE_MANAGER` role is now removed from UserRole.

- The attribute `priceTier` is now removed from List All Price Points for an In-App Purchase.
