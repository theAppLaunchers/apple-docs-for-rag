# AlternativeDistributionPackageCreateRequest

The request body you use to create an alternative distribution package.

App Store Connect API 3.3+

``` source
object AlternativeDistributionPackageCreateRequest
```

## Properties

`data`

AlternativeDistributionPackageCreateRequest.Data

 (Required) 

## Discussion

Use this object to create a new alternative distribution package in App Store Connect. For more information about the request that includes this request body, see Create an alternative distribution package.
