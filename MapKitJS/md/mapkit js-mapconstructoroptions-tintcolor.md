# tintColor

The CSS color that MapKit JS uses for the user interface controls on the map.

MapKit JS 5.0+

``` source
attribute string tintColor;
```

## Discussion

Use tintColor to color the outline of the controls, text, and glyphs. Use any valid CSS color value.
