# minimumSpan

The minimum longitudinal and latitudinal span the map displays.

MapKit JS 5.0+

``` source
attribute mapkit.CoordinateSpan minimumSpan;
```

## Discussion

Set this value to ensure that showItems sets a map region no smaller than the minimum span specified.
