# toCoordinateRegion

Returns the coordinate region that corresponds to the calling bounding region.

MapKit JS 5.0+

``` source
mapkit.CoordinateRegion toCoordinateRegion();
```

## Return Value

A mapkit.CoordinateRegion that corresponds to the calling mapkit.BoundingRegion.
