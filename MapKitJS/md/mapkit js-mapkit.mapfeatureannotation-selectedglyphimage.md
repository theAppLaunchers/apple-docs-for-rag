# selectedGlyphImage

The glyph image for the selected feature.

MapKit JS 5.74+

``` source
attribute mapkit.MapFeatureAnnotationGlyphImage selectedGlyphImage;
```

## Discussion

Note

This property is only available when the featureType is PointOfInterest.
