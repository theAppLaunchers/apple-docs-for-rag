# Adaptive

A constant indicating the measurement system is adaptive, and determined based on the map’s language.

MapKit JS 5.12+

``` source
const string Adaptive;
```

## Discussion

The measurement system used to display distances, either metric or imperial, is chosen dynamically based on the map’s set language.
