# removeAnnotation

Removes an annotation from the map.

MapKit JS 5.0+

``` source
mapkit.Annotation removeAnnotation(
 mapkit.Annotation annotation
);
```

## Parameters 

`annotation`  

The annotation to remove.

## Return Value

Returns the annotation.
