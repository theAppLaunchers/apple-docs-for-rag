# removeOverlays

Removes multiple overlays from the map.

MapKit JS 5.0+

``` source
mapkit.Overlay[] removeOverlays(
 mapkit.Overlay[] overlays
);
```

## Parameters 

`overlays`  

An array of overlays to remove.

## Return Value

Returns the array of overlays.
