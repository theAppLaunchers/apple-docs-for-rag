# equals

Returns a Boolean value that indicates whether two spans are equal.

MapKit JS 5.0+

``` source
boolean equals(
 mapkit.CoordinateSpan anotherSpan
);
```

## Parameters 

`anotherSpan`  

The span to compare.

## Return Value

`true` if the span specified by `anotherSpan` equals this span; otherwise, `false`.
