# padding

Spacing to add around the annotation when showing items.

MapKit JS 5.16+

``` source
attribute mapkit.Padding padding;
```

## Discussion

Padding prevents any items from touching the edges of the map. Set `padding` to a mapkit.Padding object (the default is no padding on all sides).
