# expectedTravelTime

The estimated travel time in seconds, including delays due to traffic.

MapKit JS 5.46+

``` source
attribute number expectedTravelTime;
```

## Discussion

The expectedTravelTime reflects the estimated time, in seconds, it takes to complete the route including traffic delay estimates.
