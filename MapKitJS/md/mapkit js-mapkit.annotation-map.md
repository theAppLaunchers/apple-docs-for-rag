# map

The map that the framework adds the annotation to.

MapKit JS 5.0+

``` source
readonly attribute mapkit.Map map;
```

## Discussion

This property is `null` before MapKit JS adds the annotation to a mapkit.Map, and after MapKit JS removes the annotation from a `mapkit.Map`.
