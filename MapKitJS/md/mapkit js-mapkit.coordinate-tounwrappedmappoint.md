# toUnwrappedMapPoint

Returns the unwrapped map point that corresponds to the coordinate.

MapKit JS 5.0+

``` source
mapkit.MapPoint toUnwrappedMapPoint();
```

## Return Value

The unwrapped map point value that corresponds to the coordinate on a two-dimensional map projection.
