# countryCode

The country or region associated with the place.

MapKit JS 5.0+

``` source
attribute string countryCode;
```

## Discussion

The country or region associated with this place, denoted by the standard abbreviation used to refer to the country or region. For example, the countryCode for the Golden Gate Park is “`US”`.
