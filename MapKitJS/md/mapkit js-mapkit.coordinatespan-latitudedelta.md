# latitudeDelta

The amount of north-to-south distance (in degrees) to display for the map region.

MapKit JS 5.0+

``` source
attribute number latitudeDelta;
```

## Discussion

Unlike longitudinal distances, which vary based on the latitude, one degree of latitude is always approximately 111 kilometers (69 miles).
