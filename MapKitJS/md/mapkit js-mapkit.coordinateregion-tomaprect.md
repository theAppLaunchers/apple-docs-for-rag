# toMapRect

Returns the map rectangle that corresponds to the calling coordinate region.

MapKit JS 5.0+

``` source
mapkit.MapRect toMapRect();
```

## Return Value

A mapkit.MapRect that corresponds to the calling mapkit.CoordinateRegion.
