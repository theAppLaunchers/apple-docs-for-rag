Instance Property

# showsPointsOfInterest

A Boolean value that determines whether the map displays points of interest.

MapKit JS 5.0+

``` source
attribute boolean showsPointsOfInterest;
```

## Discussion

This property determines whether the map shows points of interest on the map in standard and hybrid modes. A point of interest can be an airport, park, museum, hospital, and so on. The map represents a point of interest with its name and a color-coded icon.
