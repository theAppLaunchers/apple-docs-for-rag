# transportType

The mode of transportation the server uses when estimating arrival times.

MapKit JS 5.46+

``` source
attribute mapkit.Directions.Transport transportType;
```

## Discussion

Use transportType to specify the mode of transportation for your route and eta requests.

The default value of this property is Automobile.
