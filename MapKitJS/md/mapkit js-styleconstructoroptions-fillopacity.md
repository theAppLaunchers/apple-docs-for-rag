# fillOpacity

The opacity of the fill color.

MapKit JS 5.0+

``` source
attribute number fillOpacity;
```

## Discussion

The value of this property can be in the range of `0` (fully transparent) to `1` (fully opaque). The default fill opacity is `0.1`.
