# mapkit.PlaceDetail.ColorSchemes

A set of color schemes for displaying place information.

MapKit JS 5.78.1+

``` source
interface mapkit.PlaceDetail.ColorSchemes {
 const string Light;
 const string Dark;
 const string Adaptive;
};
```

## Topics

### Place Detail color scheme values

Adaptive

A constant indicating a color scheme that follows user’s current system setting.

Dark

A constant indicating a dark color scheme.

Light

A constant indicating a light color scheme.
