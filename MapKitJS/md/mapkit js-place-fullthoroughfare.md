# fullThoroughfare

A combination of thoroughfare and subthoroughfare.

MapKit JS 5.41.1+

``` source
attribute string? fullThoroughfare;
```

## Discussion

For example, the fullThoroughfare for the Golden Gate Park is `“1054 John F Kennedy Dr”`.
