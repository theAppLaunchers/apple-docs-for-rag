# showsScale

A feature visibility setting that allows you to determine when to display the map’s scale.

MapKit JS 5.0+

``` source
attribute string showsScale;
```

## Discussion

The default value is `mapkit.FeatureVisibility.Hidden`. See mapkit.FeatureVisibility for other values that determine when the map shows the scale. See mapkit’s showsScale for more information about how the values determine when to show the scale.
