# addOverlay

Adds an overlay to the map.

MapKit JS 5.0+

``` source
mapkit.Overlay addOverlay(
 mapkit.Overlay overlay
);
```

## Parameters 

`overlay`  

The overlay to add.

## Return Value

Returns the overlay.

## Discussion

MapKit JS adds the overlay to the end of the map’s overlays array.
