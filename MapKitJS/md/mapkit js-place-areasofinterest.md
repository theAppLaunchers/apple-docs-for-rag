# areasOfInterest

Common names of the area in which the place resides.

MapKit JS 5.41.1+

``` source
attribute string[]? areasOfInterest;
```

## Discussion

For example, the areasOfInterest for the Golden Gate Park are `[”Lindley Meadow”, “Golden Gate Park”]`.
