# lineWidth

The width of a line’s stroke, in CSS pixels.

MapKit JS 5.0+

``` source
attribute number lineWidth;
```

## Discussion

The default line width is `1`.

lineCap

lineDash

An array of line and gap lengths for creating a dashed line.

lineDashOffset
