# showsCompass

A feature visibility setting that determines when the compass is visible.

MapKit JS 5.0+

``` source
attribute string showsCompass;
```

## Discussion

The default value is `mapkit.FeatureVisibility.Adaptive`. See mapkit.FeatureVisibility for other settings.
