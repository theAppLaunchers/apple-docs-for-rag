# pointOfInterestCategory

The point-of-interest category of the feature.

MapKit JS 5.74+

``` source
attribute mapkit.PointOfInterestCategory pointOfInterestCategory;
```

## Discussion

Note

This property is only available when the featureType is PointOfInterest.
