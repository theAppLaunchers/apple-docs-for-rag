# alternateIds

A list of alternate Place IDs referencing a feature.

MapKit JS 5.78.1+

``` source
attribute string[]? alternateIds;
```

## Discussion

Associate alternate identifiers with a place that changes its identity over time. For information about identifiers, see Identifying unique locations with Place IDs.
