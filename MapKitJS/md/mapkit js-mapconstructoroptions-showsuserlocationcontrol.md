# showsUserLocationControl

A Boolean value that determines whether the user location control is visible.

MapKit JS 5.0+

``` source
attribute boolean showsUserLocationControl;
```

## Discussion

The default value is `false`.

### Enabling user location

showsUserLocation

tracksUserLocation
