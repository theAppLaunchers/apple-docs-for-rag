# center

The center point of the region.

MapKit JS 5.0+

``` source
attribute mapkit.Coordinate center;
```

## Discussion

Create a mapkit.Coordinate with a latitude and longitude pair:

```
var center = new mapkit.Coordinate(37.415, -122.048333); // latitude, longitude
```
