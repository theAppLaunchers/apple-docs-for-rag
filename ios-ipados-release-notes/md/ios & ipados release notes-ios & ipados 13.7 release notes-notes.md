# iOS & iPadOS 13.7 Release Notes

Update your apps to use new features, and test your apps against API changes.

## Overview

The iOS & iPadOS 13.7 SDK provides support to develop apps for iPhone, iPad, and iPod touch devices running iOS & iPadOS 13.7. The SDK comes bundled with Xcode 11.7, available from the Mac App Store. For information on the compatibility requirements for Xcode 11.7, see Xcode 11.7 Release Notes.

### General

There are no new release notes for this software update.
