# iOS & iPadOS 14.4 Release Notes

Update your apps to use new features, and test your apps against API changes.

## Overview

The iOS & iPadOS 14.4 SDK provides support to develop apps for iPhone, iPad, and iPod touch devices running iOS & iPadOS 14.4. The SDK comes bundled with Xcode 12.4, available from the Mac App Store. For information on the compatibility requirements for Xcode 12.4, see Xcode 12.4 Release Notes.
