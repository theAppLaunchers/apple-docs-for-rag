# ==(\_:\_:)

Returns a Boolean value indicating whether two values are equal.

macOS 11.0+

## Parameters 

`lhs`  

A value to compare.

`rhs`  

Another value to compare.

## Discussion

Equality is the inverse of inequality. For any values `a` and `b`, `a == b` implies that `a != b` is `false`.
