# init(directory:)

Creates a directory share with a directory that you specify on the host.

macOS 12.0+

``` source
init(directory: VZSharedDirectory)
```

## Parameters 

`directory`  

The directory to share.

## Discussion

Use this method to share a directory on the host system with the guest VM.
