# init(widthInPixels:heightInPixels:)

Creates a Virtio graphics device with the specified dimensions.

macOS 13.0+

``` source
init(
    widthInPixels: Int,
    heightInPixels: Int
)
```

## Parameters 

`widthInPixels`  

The graphics device width in pixels.

`heightInPixels`  

The graphics device height in pixels.
