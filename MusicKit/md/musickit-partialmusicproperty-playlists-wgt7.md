# playlists

An identifier for the relationship property that returns the associated playlists for the radio show.

iOS 15.4+iPadOS 15.4+Mac Catalyst 15.4+macOS 12.3+tvOS 15.4+visionOS 1.0+watchOS 9.0+

``` source
static let playlists: MusicRelationshipProperty
```

Available when `Root` is `RadioShow`.
