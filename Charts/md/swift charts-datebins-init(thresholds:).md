# init(thresholds:)

Creates N-1 bins with the given N `thresholds`.

iOS 16.0+iPadOS 16.0+Mac Catalyst 16.0+macOS 13.0+tvOS 16.0+visionOS 1.0+watchOS 9.0+

``` source
nonisolated
init(thresholds: [Date])
```

## Parameters 

`thresholds`  

The threshold values that surround the bins.

## Return Value

The bins.
