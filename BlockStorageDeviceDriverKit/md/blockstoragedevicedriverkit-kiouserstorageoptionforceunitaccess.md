# kIOUserStorageOptionForceUnitAccess

An option to use Force Unit Access (FUA).

DriverKit 21.0+

``` source
#define kIOUserStorageOptionForceUnitAccess
```

## Discussion

This option tells the storage stack to force a request to access the media.
