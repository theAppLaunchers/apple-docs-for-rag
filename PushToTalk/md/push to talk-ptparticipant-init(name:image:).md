# init(name:image:)

Creates a participant with the name and image you specify.

iOS 16.0+iPadOS 16.0+Mac Catalyst 16.0+

``` source
init(
    name: String,
    image: UIImage?
)
```

## Parameters 

`name`  

The name of the participant.

`image`  

The image you associate with the participant.
