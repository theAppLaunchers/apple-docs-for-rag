# url

Returns the URL associated with the URL action.

iOS 11.0+iPadOS 11.0+Mac Catalyst 13.1+macOS 10.5+tvOS 11.0+visionOS 1.0+

``` source
var url: URL? { get set }
```

## Return Value

The URL associated with the action, or `NULL` if no URL is specified.
