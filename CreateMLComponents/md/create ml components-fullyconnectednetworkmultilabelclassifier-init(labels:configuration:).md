# init(labels:configuration:)

Creates a full-connected network multi-label classifier.

iOS 17.0+iPadOS 17.0+Mac Catalyst 17.0+macOS 14.0+tvOS 17.0+visionOS 1.0+

``` source
init(
    labels: Set,
    configuration: FullyConnectedNetworkConfiguration = defaultConfiguration
)
```

## Parameters 

`labels`  

The labels used to train the multi-label classifier.

`configuration`  

The configuration.
