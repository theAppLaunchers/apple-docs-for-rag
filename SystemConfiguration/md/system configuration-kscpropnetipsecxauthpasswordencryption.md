# kSCPropNetIPSecXAuthPasswordEncryption

The IPSec key XAuthPasswordEncryption.

macOS 10.6+

``` source
let kSCPropNetIPSecXAuthPasswordEncryption: CFString
```

## Discussion

The value of this key is of type CFString. This key can accept the following constants:

- kSCValNetIPSecXAuthPasswordEncryptionKeychain, which has the value `Keychain`

- kSCValNetIPSecXAuthPasswordEncryptionPrompt, which has the value `Prompt`
