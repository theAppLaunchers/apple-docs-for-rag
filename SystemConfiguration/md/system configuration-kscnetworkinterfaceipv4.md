# kSCNetworkInterfaceIPv4

A network interface that can used for layering other interfaces (for example, 6to4, PPTP, or L2TP) over an existing IPv4 network.

macOS 10.4+

``` source
let kSCNetworkInterfaceIPv4: SCNetworkInterface
```
