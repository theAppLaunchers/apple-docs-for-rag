# init(highlight:trigger:)

Creates and initializes a membership event.

iOS 16.0+iPadOS 16.0+Mac Catalyst 16.0+macOS 13.0+visionOS 1.0+

``` source
init(
    highlight: SWHighlight,
    trigger: SWHighlightMembershipEventTrigger
)
```

## Parameters 

`highlight`  

The related hightlight for the event.

`trigger`  

The trigger membership event for the highlight.
