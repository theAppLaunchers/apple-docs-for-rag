# Dictionary Keys

Keys to use when working with dictionary representations of a time range.

## Overview

To convert a `CMTimeRange` to and from a `CFDictionary`, see CMTimeRangeCopyAsDictionary(_:allocator:) and CMTimeRangeMakeFromDictionary(_:).

## Topics

### Constants

let kCMTimeRangeStartKey: CFString

The key for the start field of a time range.

let kCMTimeRangeDurationKey: CFString

The key for the timescale of a time range.
