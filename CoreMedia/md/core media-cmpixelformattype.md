# CMPixelFormatType

A pixel format type.

iOS 4.0+iPadOS 4.0+Mac Catalyst 13.1+macOS 10.7+tvOS 9.0+visionOS 1.0+watchOS 6.0+

``` source
typealias CMPixelFormatType = FourCharCode
```

## Discussion

Only some codec types are pixel formats.
