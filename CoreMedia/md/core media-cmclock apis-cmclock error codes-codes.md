# CMClock Error Codes

Constants that represent the errors in Core Media clock operations.

## Topics

### Constants

var kCMClockError_MissingRequiredParameter: OSStatus

A clock error that indicates a parameter is missing.

var kCMClockError_InvalidParameter: OSStatus

A clock error that indicates a parameter isn’t valid.

var kCMClockError_AllocationFailed: OSStatus

A clock error that indicates the memory allocation fails.

var kCMClockError_UnsupportedOperation: OSStatus

A clock error that indicates the operation isn’t supported.
