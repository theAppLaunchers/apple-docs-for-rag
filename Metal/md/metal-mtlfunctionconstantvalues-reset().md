# reset()

Deletes all previously set constant values.

iOS 10.0+iPadOS 10.0+Mac Catalyst 13.1+macOS 10.12+tvOS 10.0+visionOS 1.0+

``` source
func reset()
```

## Discussion

You do not need to call this method if you have not set any constant values.
