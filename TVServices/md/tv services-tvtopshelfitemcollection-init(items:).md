# init(items:)

Creates an item collection object from the specified set of top shelf items.

tvOS 13.0+

``` source
init(items: [Item])
```

## Parameters 

`items`  

The array of TVTopShelfItem objects to put in the collection.

## Return Value

A new item collection object containing the specified `items`.
