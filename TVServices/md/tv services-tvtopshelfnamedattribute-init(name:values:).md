# init(name:values:)

Creates a new named attribute object with the specified values.

tvOS 13.0+

``` source
init(
    name: String,
    values: [String]
)
```

## Parameters 

`name`  

The localized name of the attribute.

`values`  

An array of strings containing the attribute values.

## Return Value

A new named attribute object.
