# namedAttributes

Additional information to display for your content, such as a list of leading actors.

tvOS 13.0+

``` source
var namedAttributes: [TVTopShelfNamedAttribute] { get set }
```

## Discussion

Use this property to specify additional details about your content. In appropriate views, the system formats the provided information and displays it alongside the rest of your content. For information about how to create named attributes, see TVTopShelfNamedAttribute.
