# userInterfaceStyleDark

An image meant for a dark user interface.

tvOS 11.0+

``` source
static var userInterfaceStyleDark: TVContentItemImageTrait { get }
```

## Discussion

Specify this constant, or the userInterfaceStyleLight constant, but not both.
