# imageSize

The standard width and height for images in an inset interface.

tvOS 13.0+

``` source
class var imageSize: CGSize { get }
```

## Discussion

The standard image size for inset items is 1740 x 560 points. For more information about configuring the images for an inset interface, see Human Interface Guidelines.
