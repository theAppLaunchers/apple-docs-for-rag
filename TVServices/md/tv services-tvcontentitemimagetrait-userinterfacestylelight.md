# userInterfaceStyleLight

An image meant for a light user interface.

tvOS 11.0+

``` source
static var userInterfaceStyleLight: TVContentItemImageTrait { get }
```

## Discussion

Specify this constant, or the userInterfaceStyleDark constant, but not both.
