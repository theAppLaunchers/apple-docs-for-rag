# swipeUp(velocity:)

Sends a swipe-up gesture with a velocity you specify.

macOSXcode 16.3+

``` source
@MainActor
func swipeUp(velocity: XCUIGestureVelocity)
```

## Parameters 

`velocity`  

The speed at which to perform the swipe-up gesture, in pixels per second.
