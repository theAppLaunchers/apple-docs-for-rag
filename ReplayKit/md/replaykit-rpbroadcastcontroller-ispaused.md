# isPaused

A Boolean value indicating whether the broadcast is paused.

iOS 10.0+iPadOS 10.0+Mac Catalyst 13.1+macOS 11.0+tvOS 10.0+visionOS 1.0+

``` source
var isPaused: Bool { get }
```

## Discussion

When set to true, the broadcast is paused.
