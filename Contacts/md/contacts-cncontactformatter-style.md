# style

The formatting style for the contact name.

iOS 9.0+iPadOS 9.0+Mac Catalyst 13.1+macOS 10.11+visionOS 1.0+watchOS 2.0+

``` source
var style: CNContactFormatterStyle { get set }
```

## Discussion

The default value for this property is CNContactFormatterStyle.fullName. For more information on formatting styles, see CNContactFormatterStyle.
