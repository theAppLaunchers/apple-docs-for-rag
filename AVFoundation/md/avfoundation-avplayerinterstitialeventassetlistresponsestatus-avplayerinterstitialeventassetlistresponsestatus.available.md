# AVPlayerInterstitialEventAssetListResponseStatus.available

Indicates that a valid asset list response is available.

iOS 16.4+iPadOS 16.4+Mac Catalyst 16.4+macOS 13.3+tvOS 16.4+visionOS 1.0+watchOS 9.4+

``` source
case available
```

## Discussion

You can retrieve the value by querying the assetListResponse property of the interstitial event.

## See Also

### Status values
