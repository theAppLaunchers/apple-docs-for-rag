# legible

A media characteristic that indicates that a track or media selection option includes legible content.

iOS 4.0+iPadOS 4.0+Mac Catalyst 13.1+macOS 10.7+tvOS 9.0+visionOS 1.0+watchOS 1.0+

``` source
static let legible: AVMediaCharacteristic
```

## Mentioned in 

Selecting Subtitles and Alternative Audio Tracks

## Discussion

Media types with this characteristic include subtitle and closedCaption.
