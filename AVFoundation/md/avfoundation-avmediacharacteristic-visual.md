# visual

A media characteristic that indicates that a track or media selection option includes visual content.

iOS 4.0+iPadOS 4.0+Mac Catalyst 13.1+macOS 10.7+tvOS 9.0+visionOS 1.0+watchOS 1.0+

``` source
static let visual: AVMediaCharacteristic
```

## Mentioned in 

Selecting Subtitles and Alternative Audio Tracks

## Discussion

Media types with this characteristic include video, subtitle, and closedCaption.
