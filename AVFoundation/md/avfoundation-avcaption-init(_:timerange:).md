# init(\_:timeRange:)

Creates a caption that contains text and a time range.

iOS 18.0+iPadOS 18.0+Mac Catalyst 15.0+macOS 12.0+

``` source
init(
    _ text: String,
    timeRange: CMTimeRange
)
```

## Parameters 

`text`  

The text the caption displays.

`timeRange`  

The range of time when the caption is active.
