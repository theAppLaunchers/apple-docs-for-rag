# mp3

The UTI for the MPEG Audio Layer III file format.

iOS 7.0+iPadOS 7.0+Mac Catalyst 13.1+macOS 10.9+tvOS 9.0+visionOS 1.0+watchOS 1.0+

``` source
static let mp3: AVFileType
```

## Discussion

The value of this UTI is `public.mp3`. Files of this type have an `.mp3` extension.
