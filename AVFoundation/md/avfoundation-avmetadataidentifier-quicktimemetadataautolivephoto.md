# quickTimeMetadataAutoLivePhoto

An identifier that represents whether the live photo movie used auto mode.

iOS 13.0+iPadOS 13.0+Mac Catalyst 13.1+macOS 10.15+tvOS 13.0+visionOS 1.0+watchOS 6.0+

``` source
static let quickTimeMetadataAutoLivePhoto: AVMetadataIdentifier
```

## Discussion

If your app captures Live Photo content, you can add this movie-level metadata to signify that the movie content should only be used if itâ€™s sufficiently dynamic for experiences like swiping between photos or force-touching a photo. For example, photos of static scenes or documents are excluded from the Live Photo experience.
