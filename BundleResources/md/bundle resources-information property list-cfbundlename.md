# CFBundleName

A user-visible short name for the bundle.

iOS 2.0+iPadOS 2.0+macOS 10.0+tvOS 9.0+visionOS 1.0+watchOS 2.0+

## Details 

Name  
Bundle name

Type  

string

## Mentioned in 

Managing Your App’s Information Property List

## Discussion

This name can contain up to 15 characters. The system may display it to users if CFBundleDisplayName isn’t set.
