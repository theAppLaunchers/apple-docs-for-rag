# NSPrefPaneIconLabel

The name of a preference pane displayed beneath the preference pane icon in the System Preferences app.

macOS 10.0+

## Details 

Name  
Preference Pane icon label

Type  

string

## Discussion

For long names, you can split the name into two lines by including a newline character (`‘\n’`) in the string. If you omit this key, System Preferences uses the CFBundleName key for the name.
