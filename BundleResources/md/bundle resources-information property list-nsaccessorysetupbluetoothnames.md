# NSAccessorySetupBluetoothNames

An array of strings that represent the Bluetooth device names or substrings for accessories that your app configures.

iOS 18.0+iPadOS 18.0+

## Details 

Name  
AccessorySetupKit - Bluetooth Names

Type  

array of strings

## Discussion

Each entry is a full or partial name which the accessory advertises and helps AccessorySetupKit identify it.
