# NSSupportsLiveActivities

A Boolean value that indicates whether an app supports Live Activities.

iOS 16.1+iPadOS 16.1+

## Details 

Name  
Supports Live Activities

Type  

boolean

## Discussion

Set the corresponding value to `true` to enable your app to display live data on the Lock Screen and in the Dynamic Island. For more information, see Displaying live data with Live Activities.
