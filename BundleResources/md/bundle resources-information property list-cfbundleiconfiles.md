# CFBundleIconFiles

The names of the bundle’s icon image files.

iOS 3.2+iPadOS 3.2+tvOS 9.0+visionOS 1.0+watchOS 2.0+

## Details 

Name  
Icon files

Type  

array of strings

## Discussion

Note

For Action extensions, you can specify the CFBundleSymbolName key instead of `CFBundleIconFiles` to use a symbol from SF Symbols.
