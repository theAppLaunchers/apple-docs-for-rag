# WeatherKit Entitlement

A Boolean value that indicates whether the app may use WeatherKit.

iOS 16.0+iPadOS 16.0+Mac Catalyst 16.0+macOS 13.0+tvOS 16.0+watchOS 9.0+

## Details 

Key  
com.apple.developer.weatherkit

Type  

boolean

## Attributes 

Default: `NO`

## Discussion

To add this entitlement to your app, enable the WeatherKit capability in Xcode.
