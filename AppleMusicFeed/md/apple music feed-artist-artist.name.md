# Artist.Name

A mapping of locale to localized names for the artist.

AppleMusicFeed 1.0+

``` source
object Artist.Name
```

## Properties

`locale`

`string`

A locale in the IETF language tag format. An empty value indicates fallback to `default`.

`name`

`string`

A localized uncensored name for the artist.
