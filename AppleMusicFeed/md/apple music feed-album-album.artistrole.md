# Album.ArtistRole

Information about an artist’s role.

AppleMusicFeed 1.0+

``` source
object Album.ArtistRole
```

## Properties

`artistId`

`string`

The identifier for the artist in Apple Music.

`artistName`

`string`

The name of the artist.

`roleName`

`string`

The artist’s role. Possible values include `compose-author`, `contributor`, `translator`, `performer`, `arranger`, `composer`, `adapter`, and `author`.
