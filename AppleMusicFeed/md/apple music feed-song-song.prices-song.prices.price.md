# Song.Prices.Price

Information about a pricing offer.

AppleMusicFeed 1.0+

``` source
object Song.Prices.Price
```

## Properties

`currencyCode`

`string`

The currency for the offer in ISO 4217 format.

`price`

`number`

The price for the offer.

`priceType`

`string`

The type of content availability for the offer. The possible values are `buy`, `streaming`, and `preorder`.

`quality`

`string`

The audio quality for the offer. The possible values are `standard-definition` and `high-definition`.
