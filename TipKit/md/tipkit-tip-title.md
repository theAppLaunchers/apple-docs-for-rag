# title

A title that names the tip.

iOS 17.0+iPadOS 17.0+Mac Catalyst 17.0+macOS 14.0+tvOS 17.0+visionOS 1.0+watchOS 10.0+

``` source
var title: Text { get }
```

**Required**

## Discussion

Use this property to convey the benefit of your tip. All tips require a title.
