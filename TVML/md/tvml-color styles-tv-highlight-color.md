# tv-highlight-color

Changes an element’s color when it comes into focus.

## Overview

Here’s an example of how to change a button’s text color when the button comes into focus.

```

   Push me

```

### Values for tv-highlight-color

`rgb(r,g,b)`  
The background color of an element, where `r,g,b` are the red, green, and blue color components, and have a value range from `0-255`.

`rgba(r,g,b,a)`  
The background color of an element, where `r,g,b` are the red, green, and blue color components, and have a value range from `0-255`, and `a` is the alpha applied to the color, ranging from `0.0-1.0`.

`transparent`  
The background color of the element, which is transparent.

### Elements that Use tv-highlight-color

- badge

- decorationLabel

- description

- listItemLockup

- menuItem

- ordinal

- ratingCard

- reviewCard

- segmentBar

- subtitle

- text

- title
