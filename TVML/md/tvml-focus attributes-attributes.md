# Focus Attributes

Define how an element acts when it comes into focus.

## Topics

### Focus Attributes

autoHighlight

Specifies that the element should initially be in focus.

disabled

Sets an element as unfocusable.

handlesOverflow

Adds the “MORE” label to a description.
