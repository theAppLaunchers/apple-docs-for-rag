# background-color

Changes the background color of an element.

## Overview

Here’s an example of how to change the background color of a `description` element.

```
Light purple background
```

### Values for background-color

`rgb(r,g,b)`  
The background color of an element, where `r,g,b` are the red, green, and blue color components, and have a value range from `0-255`.

`rgba(r,g,b,a)`  
The background color of an element, where `r,g,b` are the red, green, and blue color components, and have a value range from `0-255`, and `a` is the alpha applied to the color, ranging from `0.0-1.0`.

`transparent`  
The background color of the element, which is transparent.

### Elements that Use background-color

- description

- ratingCard

- reviewCard

- textBadge
