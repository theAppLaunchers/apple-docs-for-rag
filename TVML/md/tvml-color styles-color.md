# color

Changes the color of an element.

## Overview

Use the `color` style to change the color an an element. Here’s an example of how to change the text color inside of a `description` element.

```
Light purple
```

### Values for color

`rgb(r,g,b)`  
The background color of an element, where `r,g,b` are the red, green, and blue color components, and have a value range from `0-255`.

`rgba(r,g,b,a)`  
The background color of an element, where `r,g,b` are the red, green, and blue color components, and have a value range from `0-255`, and `a` is the alpha applied to the color, ranging from `0.0-1.0`.

`transparent`  
The background color of the element is transparent.

### Elements that Use color

- decorationLabel

- description

- ordinal

- segmentBar

- subtitle

- text

- textBadge

- title
