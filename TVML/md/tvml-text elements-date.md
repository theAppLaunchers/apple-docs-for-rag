# date

Displays a date.

## Overview

The `date` element contains text used to display the associated date. The date string must be in one of the following formats:

- yyyy-MM-dd

- ISO date format

- RFC 1123 date format

Here’s an example that sets the date to January 19, 2031.

```
2031-01-19
```
