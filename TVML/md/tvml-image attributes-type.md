# type

Specifies how a badge is drawn.

## Overview

Use the `type` attribute to specify whether a badge is filled or an outline.

### Values for type

`fill`  
Completely fills the badge icon.

`stroke`  
Draws the badge icon as an outline. This is the default value.

### Elements that Use type

- textBadge
