# segmentBarItem

Provides titles inside of a segment bar.

## Overview

The `segmentBarItem` element is used to create groups inside of a `segmentBarHeader`. Users can move between `segmentBarItem` elements to display related information. Hereâ€™s an example that has a title and subtitle above a segment bar that contains one item.
