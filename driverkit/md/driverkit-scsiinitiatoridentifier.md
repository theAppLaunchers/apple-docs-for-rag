# SCSIInitiatorIdentifier

64-bit number to represent a SCSI Initiator Device.

DriverKitiOSiPadOSmacOS


## Discussion

If the identifier is for an initiator only and not a target, then SCSIInitiatorIdentifier should be used.
