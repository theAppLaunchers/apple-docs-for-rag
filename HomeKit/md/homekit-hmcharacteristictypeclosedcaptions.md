# HMCharacteristicTypeClosedCaptions

An indictator of whether closed captions are enabled or disabled.

iOS 18.0+iPadOS 18.0+Mac Catalyst 18.0+tvOS 18.0+visionOS 2.0+watchOS 11.0+

``` source
let HMCharacteristicTypeClosedCaptions: String
```

## Overview

The corresponding value is one of the constants in the HMCharacteristicValueClosedCaptions enumeration.

## Topics

### Values

enum HMCharacteristicValueClosedCaptions

The state for closed captions.
