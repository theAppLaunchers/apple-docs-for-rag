# ChairType

Types of chair that the framework identifies in a captured room.

iOS 17.0+iPadOS 17.0+Mac Catalyst 17.0+

``` source
enum ChairType
```

## Overview

When the framework observes a chair in the physical environment during a scan, it chooses a type among these options that best matches the chairâ€™s look. The framework adds that instance of this enum to the attributes array for the object (see objects) that represents the chair in the captured room.
