# init(mediaTypes:)

Initializes a media item picker for specified media types.

iOS 3.0+iPadOS 3.0+Mac Catalyst 13.1+

``` source
@MainActor
init(mediaTypes: MPMediaType)
```

## Parameters 

`mediaTypes`  

A set representing the media types for the media item picker. See the MPMediaType enumeration.

## Return Value

A new media item picker.
