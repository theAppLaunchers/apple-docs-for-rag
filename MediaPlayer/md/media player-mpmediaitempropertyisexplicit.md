# MPMediaItemPropertyIsExplicit

A Boolean value that indicates whether the media item contains explicit (adult) lyrics or language.

iOS 7.0+iPadOS 7.0+Mac Catalyst 13.1+macOS 10.12.2+tvOS 9.0+visionOS 1.0+watchOS 5.0+

``` source
let MPMediaItemPropertyIsExplicit: String
```

## Discussion

This value is an NSNumber object that represents a `BOOL` data type.
