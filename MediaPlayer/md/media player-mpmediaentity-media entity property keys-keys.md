# Media entity property keys

The property keys used to retrieve metadata for media entities.

## Overview

You obtain metadata for a media entity by calling the value(forProperty:) method with these property keys. You can also use some properties to build media property predicates, as described in MPMediaPropertyPredicate. These properties are filterable.
