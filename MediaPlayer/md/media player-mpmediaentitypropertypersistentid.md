# MPMediaEntityPropertyPersistentID

The persistent identifier for a media entity.

iOS 4.2+iPadOS 4.2+Mac Catalyst 13.1+macOS 10.12.2+tvOS 9.0+visionOS 1.0+

``` source
let MPMediaEntityPropertyPersistentID: String
```

## Discussion

This value contains a uint64_t (unsigned long long) which you can use to build a media property predicate, as described in MPMediaPropertyPredicate.
