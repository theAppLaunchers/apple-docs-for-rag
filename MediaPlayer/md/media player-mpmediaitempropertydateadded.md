# MPMediaItemPropertyDateAdded

The date the media item was added to the user’s Media library.

iOS 10.0+iPadOS 10.0+Mac Catalyst 13.1+macOS 10.12.2+tvOS 10.0+visionOS 1.0+watchOS 5.0+

``` source
let MPMediaItemPropertyDateAdded: String
```

## Discussion

Value is a NSDate object.
