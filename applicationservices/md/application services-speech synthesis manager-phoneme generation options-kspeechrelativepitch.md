# kSpeechRelativePitch

macOS 10.6+

``` source
var kSpeechRelativePitch: Int32 { get }
```

## Discussion

Specify the pitch in the tune, relative to the pitch baseline of the voice (this makes it easier to reuse the tune with a different voice).
