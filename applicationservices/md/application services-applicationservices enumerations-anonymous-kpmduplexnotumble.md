# kPMDuplexNoTumble

macOS 10.0+

``` source
var kPMDuplexNoTumble: Int { get }
```

## Discussion

Print on both sides of the paper, with both sides oriented in the same direction (no tumbling). This is equivalent to choosing "long edge binding" in the Print pane.
