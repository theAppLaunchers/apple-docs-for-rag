# Speech Synthesizer Information Keys

Keys used with the `kSpeechSynthesizerInfoProperty` property to get information about the synthesizer.

## Topics

### Constants

let kSpeechSynthesizerInfoIdentifier: CFString

The identifier of the speech synthesizer.

let kSpeechSynthesizerInfoVersion: CFString

The version of the speech synthesizer.

let kSpeechSynthesizerInfoManufacturer: CFString

The identification of a synthesizer engine (see the `synthManufacturer` field in SpeechVersionInfo).
