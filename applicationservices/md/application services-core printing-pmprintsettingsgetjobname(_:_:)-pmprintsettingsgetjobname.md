Function

# PMPrintSettingsGetJobName(\_:\_:)

Obtains the name of a print job.

macOS 10.4+

``` source
func PMPrintSettingsGetJobName(
    _ printSettings: PMPrintSettings,
    _ name: UnsafeMutablePointer?>
) -> OSStatus
```

## Parameters 

`printSettings`  

The print settings for the current print job.

`name`  

A pointer to your CFString variable. On return, the variable refers to a Core Foundation string containing the name of the print job. This is the same job name you set using the function PMPrintSettingsSetJobName(_:_:). You should not release the string without first retaining it.

## Return Value

A result code. See Result Codes.
