# ==(\_:\_:)

Returns a Boolean value indicating whether two values are equal.

iOS 18.0+

``` source
static func == (a: JournalingSuggestion.MotionActivity.MovementType, b: JournalingSuggestion.MotionActivity.MovementType) -> Bool
```

## Parameters 

`lhs`  

A value to compare.

`rhs`  

Another value to compare.

## Discussion

Equality is the inverse of inequality. For any values `a` and `b`, `a == b` implies that `a != b` is `false`.
