# subtitle

The subtitle string to display for the group of annotations.

iOS 11.0+iPadOS 11.0+Mac Catalyst 13.1+macOS 10.13+tvOS 11.0+visionOS 1.0+

``` source
var subtitle: String? { get set }
```

## Discussion

The cluster annotation object sets this property to a string that conveys how many additional annotations the group includes, except for the annotation with the title that the title property displays.
