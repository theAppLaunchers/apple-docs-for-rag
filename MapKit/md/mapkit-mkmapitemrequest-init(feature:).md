# init(feature:)

Creates a new map item request with the specified map feature.

MapKitSwiftUIiOS 17.0+iPadOS 17.0+Mac CatalystvisionOS

``` source
convenience init(feature: MapFeature)
```

## Parameters 

`feature`  

The MapFeature to use to initialize the map item request.
