# init(scope:)

Creates a new user location button with the scope you specify.

MapKitSwiftUIiOS 17.0+iPadOS 17.0+Mac CatalystmacOS 14.0+tvOS 17.0+visionOSwatchOS 10.0+

``` source

init(scope: Namespace.ID? = nil)
```

## Parameters 

`scope`  

A Namespace.ID value that identifies this namespace and that you use to associate the user location indicator with a map instance.
