# init(origin:size:)

Creates the map rectangle with the specified point and size.

iOSiPadOSMac CatalystmacOStvOSvisionOSwatchOS

``` source
init(
    origin: MKMapPoint,
    size: MKMapSize
)
```

## Parameters 

`origin`  

The origin of the map rectangle as an MKMapPoint.

`size`  

The size of the map rectangle as an MKMapSize.
