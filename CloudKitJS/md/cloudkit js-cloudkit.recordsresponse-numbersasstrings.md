# numbersAsStrings

A Boolean value indicating whether the numbers in fields are represented as strings.

CloudKit JS 1.0+

``` source
readonly attribute Boolean numbersAsStrings;
```

## Discussion

Set the `numbersAsStrings` key in the `options` parameter when requesting records using the fetchRecords and similar methods.
