# apnsEnvironment

The Apple Push Notification service (APNs) environment associated with this container.

CloudKit JS 1.0+

``` source
attribute String apnsEnvironment;
```

## Discussion

Possible values are CloudKit.DEVELOPMENT_ENVIRONMENTand CloudKit.PRODUCTION_ENVIRONMENT.
