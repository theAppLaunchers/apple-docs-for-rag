# alertLocalizationArgs

An array of strings that appear as variables if alertLocalizationKey is a format specifier.

CloudKit JS 1.0+

``` source
readonly attribute String[] alertLocalizationArgs;
```

## Discussion

For details, read Apple Push Notification Service in Local and Remote Notification Programming Guide.
