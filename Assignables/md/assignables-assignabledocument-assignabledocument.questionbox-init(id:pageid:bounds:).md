# init(id:pageID:bounds:)

Initialize a question box.

iOS 17.4+iPadOS 17.4+Mac Catalyst 17.4+visionOS

``` source
init(
    id: AssignableDocument.QuestionBox.ID,
    pageID: AssignableDocument.Page.ID,
    bounds: CGRect
)
```

## Parameters 

`id`  

An ID for the box you can use to reference it within the document.

`pageID`  

The ID of the page the box is for.

`bounds`  

The bounds of the box on the page.
