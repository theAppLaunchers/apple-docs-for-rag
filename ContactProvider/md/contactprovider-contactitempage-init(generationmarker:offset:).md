# init(generationMarker:offset:)

Creates a contact item page with the given generation marker and offset.

iOS 18.0+iPadOS 18.0+

``` source
init(
    generationMarker: Data,
    offset: Int
)
```

## Parameters 

`generationMarker`  

A marker that indicates when enumeration of content started.

`offset`  

An offset from the generation marker.
