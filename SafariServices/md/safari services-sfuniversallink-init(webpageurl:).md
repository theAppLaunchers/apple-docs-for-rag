Initializer

# init(webpageURL:)

Creates a universal link object with the URL.

macOS 10.15+

``` source
init?(webpageURL url: URL)
```

## Parameters 

`url`  

The URL of the web page to associate.

## Discussion

This designated initializer creates a SFUniversalLink object.
