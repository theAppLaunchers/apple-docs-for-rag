# ASImportableItem.ItemType.document

The type for an item that contains document-related credential information.

iOS 18.2+iPadOS 18.2+Mac Catalyst 18.2+macOS 15.2+visionOS 2.2+

``` source
case document
```

## Discussion

This type of item contains credential information like a note or file, as defined in the Credential Exchange Format (CXF) specification.
