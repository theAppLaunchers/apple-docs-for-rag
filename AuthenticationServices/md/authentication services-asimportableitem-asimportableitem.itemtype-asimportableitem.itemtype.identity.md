# ASImportableItem.ItemType.identity

The type for an item that contains identity-related credential information.

iOS 18.2+iPadOS 18.2+Mac Catalyst 18.2+macOS 15.2+visionOS 2.2+

``` source
case identity
```

## Discussion

This type of item contains credential information like a credit card number or address, as defined in the Credential Exchange Format (CXF) specification.
