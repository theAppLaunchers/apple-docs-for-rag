# AudioUnitParameterUnit.boolean

A Boolean-like unit of measure.

iOSiPadOSMac CatalystmacOStvOSvisionOS


## Discussion

A value of 0.0 means `FALSE` and a nonzero value means `TRUE`.
