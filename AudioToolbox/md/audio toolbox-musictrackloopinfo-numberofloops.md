# numberOfLoops

The number of times to play the designated portion of the music track.

iOSiPadOSMac CatalystmacOStvOSvisionOS

``` source
var numberOfLoops: Int32
```

## Discussion

To loop forever, set `numberOfLoops` to `0`.
