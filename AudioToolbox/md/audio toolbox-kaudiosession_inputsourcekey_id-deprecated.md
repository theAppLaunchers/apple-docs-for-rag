# kAudioSession_InputSourceKey_ID

An audio input source.

Mac Catalyst 14.0–14.0DeprecatedvisionOS 1.0–1.0Deprecated

``` source
let kAudioSession_InputSourceKey_ID: CFString!
```

Deprecated

Deprecated in iOS 7.0.

## Discussion

A CFNumber object, defined by a USB audio accessory attached to the device through the iPad camera connection kit, that identifies an audio input source. When setting a source on the accessory, use this identifier.
