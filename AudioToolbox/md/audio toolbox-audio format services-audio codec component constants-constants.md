# Audio Codec Component Constants

Audio codec component types.

## Overview

Some possible audio codec component subtypes are listed in the format IDs enumeration in the CoreAudioTypes.h header file (see Core Audio Data Types).

## Topics

### Constants

var kAudioDecoderComponentType: UInt32

A codec that translates data in some other format into linear PCM.

var kAudioEncoderComponentType: UInt32

A codec that translates linear PCM data into some other format
