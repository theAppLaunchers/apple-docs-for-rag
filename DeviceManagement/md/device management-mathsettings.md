# MathSettings

The declaration to configure the math and calculator apps.

iOS 18.0+iPadOS 18.0+macOS 15.0+Device Assignment ServicesVPP License Management

``` source
object MathSettings
```

## Properties

`Calculator`

MathSettingsCalculatorObject

If present, configures the built-in Calculator app.

`SystemBehavior`

MathSettingsSystemBehaviorObject

If present, configures math behavior in the system.
