# WidgetFamily.systemExtraLarge

An extra-large widget.

iOS 15.0+iPadOS 15.0+Mac Catalyst 15.0+macOS 14.0+

``` source
case systemExtraLarge
```

## Mentioned in 

Preparing widgets for additional platforms, contexts, and appearances

Adding interactivity to widgets and Live Activities

## Discussion

The extra-large system widget can appear on the Home Screen or in the Today View in iPadOS. In macOS, the extra-large system widget can appear on the desktop or in Notification Center.

Note

This value is only available in iPadOS and macOS.
