# AccessoryWidgetGroupStyle

The style for an AccessoryWidgetGroup view.

watchOS 11.0+

``` source
struct AccessoryWidgetGroupStyle
```

## Overview

Use the `View/accessoryWidgetGroupStyle(_:)` modifier to set the desired style.

## Topics

### Type Properties

static let automatic: AccessoryWidgetGroupStyle

The default style that is set to circular.

static let circular: AccessoryWidgetGroupStyle

Masks each content view with a circle.

static let roundedSquare: AccessoryWidgetGroupStyle

Masks each content view with a rounded square.
