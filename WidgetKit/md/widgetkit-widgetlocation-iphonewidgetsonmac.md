# iPhoneWidgetsOnMac

The widget originates from another device and appears on the Mac.

iOS 17.0+iPadOS 17.0+Mac Catalyst 17.0+

``` source
static let iPhoneWidgetsOnMac: WidgetLocation
```

### Specifying a location style
