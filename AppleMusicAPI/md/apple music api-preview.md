# Preview

An object that represents a preview for resources.

Apple Music 1.0+

``` source
object Preview
```

## Properties

`artwork`

Artwork

The preview artwork for the associated preview music video.

`url`

`string`

 (Required) 

The preview URL for the content.

`hlsUrl`

`string`

The HLS preview URL for the content.
