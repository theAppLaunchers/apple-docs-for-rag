# Target

An object that provides methods to obtain information about the installation volume.

## Overview

This object is accessed using the `my.target` property.

## Topics

### Accessing Target Properties

mountpoint

The installation volume’s mountpoint, as a String.

availableKilobytes

Available space in the installation volume, in kilobytes.

systemVersion

The version of the host’s operating system, as a String.

### Accessing Installation Receipts

receiptForIdentifier

Provides information about the receipt for a given package on the installation host.
