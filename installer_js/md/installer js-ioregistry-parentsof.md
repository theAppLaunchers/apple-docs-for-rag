# parentsOf

Returns the parents of the given IOKit object.

``` source
childrenOf(objectName)
```

## Parameters 

`objectName`  

String with the path of the desired IOKit object, or an item of an IOKit object array.

## Return Value

An array of IOKit object dictionaries.
