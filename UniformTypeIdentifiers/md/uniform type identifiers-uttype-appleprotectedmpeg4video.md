# appleProtectedMPEG4Video

A type that represents data in Apple-protected MPEG-4 format.

iOS 14.0+iPadOS 14.0+Mac Catalyst 14.0+macOS 11.0+tvOS 14.0+visionOS 1.0+watchOS 7.0+

``` source
static var appleProtectedMPEG4Video: UTType { get }
```

## Discussion

The identifier for this type is `com.apple.protected-mpeg-4-video`.

This type conforms to a base type identified by `com.apple.m4v-video`, which in turn conforms to UTTypeMPEG4Movie.
