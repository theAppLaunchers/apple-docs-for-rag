# heif

A type that represents High Efficiency Image File Format images.

iOS 14.0+iPadOS 14.0+Mac Catalyst 14.0+macOS 11.0+tvOS 14.0+visionOS 1.0+watchOS 7.0+

``` source
static var heif: UTType { get }
```

## Discussion

The identifier for this type is `public.heif`.

This type conforms to a base type identified by `public.heif-standard`, which in turn conforms to UTTypeImage.
