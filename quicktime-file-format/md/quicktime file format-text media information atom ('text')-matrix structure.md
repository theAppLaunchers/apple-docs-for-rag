# Matrix structure

A matrix structure associated with this text media.

## Overview

This should be the identity matrix. A matrix shows how to map points from one coordinate space into another. See Matrices for a discussion of how display matrices are used in QuickTime and see Matrix structure for an illustration of a matrix structure within an atom.

Size

A 32-bit integer that identifies the atom type.
