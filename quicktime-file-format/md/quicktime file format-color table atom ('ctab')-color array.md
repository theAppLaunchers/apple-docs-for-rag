# Color array

An array of colors.

## Overview

Each color is made of four unsigned 16-bit integers. The first integer must be set to `0`, the second is the red value, the third is the green value, and the fourth is the blue value.

## See Also

### Data fields

Size

A 32-bit integer that specifies the number of bytes in this color table atom.

Type

A 32-bit integer that identifies the atom type.

Color table seed

A 32-bit integer.

Color table flags

A 16-bit integer.

Color table size

A 16-bit integer that indicates the number of colors in the following color array.
