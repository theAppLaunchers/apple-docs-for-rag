# Track reference atom ('tref')

An atom that defines relationships between tracks.

## Mentioned in 

Referencing a related forced subtitle track

## Topics

### Data fields

Size

A 32-bit integer that specifies the number of bytes in this track reference atom.

Type

A 32-bit integer that identifies the atom type.

Track reference type atom

A list of track reference type atoms containing the track reference information.
