# Type

The type of this atom.

## Overview

This field must be set to `'rmdr'`.

## See Also

### Data fields

Size

The number of bytes in this data rate atom.

Flags

A 32-bit integer.

Data rate

The required data rate in bits per second, expressed as a 32-bit integer.
