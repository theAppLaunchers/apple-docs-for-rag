# Warning

See the properties of a warning the Apple News API returned.

Apple News API 1.0+

``` source
object Warning
```

## Properties

`keypath`

`[string]`

An array of field names that uniquely identifies a field in the JSON input of the request.

`message`

`string`

A user-friendly, detailed explanation of the nonfatal warning.

`value`

`string`

If applicable, the value supplied in the request for the field that `keyPath` specifies.
