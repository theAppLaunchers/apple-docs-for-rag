# Status

See the HTTP status codes the Apple News API returned.

Apple News API 1.0+

``` source
integer Status
```

## Possible Values 

`200`  

OK

`201`  

Created

`400`  

Bad request

`401`  

Unauthorized

`403`  

Forbidden

`404`  

Not Found

`500`  

Internal Server Error

`409`  

Conflict

`415`  

Unsupported Media Type

`429`  

Too Many Requests
