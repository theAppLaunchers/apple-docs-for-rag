# Apple News Format Release Notes for iOS 17, iPadOS 17, and macOS 14

Learn about new features that require iOS 17, iPadOS 17, and macOS 14.

## Overview

This feature was added for the latest Apple News Format release:

- You can now use WebP images for thumbnail images and in articles. For more information, see Preparing Image, Video, Audio, Music, and ARKit Assets.

This feature was added in a previous Apple News Format release:

- In the ComponentTextStyle object, use `hyphenationMinimumWordLength`, `hyphenationMinimumCharactersBefore`, `hyphenationMinimumCharactersAfter`, `hyphenationZone`, and `lineBalancing` properties to improve text clipping and line-breaking behavior in an article.

### Deprecated Objects and Properties

This property was deprecated in the latest Apple News Format release:

- The `transparentToolbar` property in the Metadata object was deprecated.

These objects were deprecated in a previous Apple News Format release:

- AutoPlacement, AdvertisementAutoPlacement, ConditionalAutoPlacement, and AutoPlacementLayout objects are deprecated because you can only control advertisement settings via News Publisher.

- MediumRectangleAdvertisement and BannerAdvertisement objects are deprecated because Apple News no longer supports manual ads.
