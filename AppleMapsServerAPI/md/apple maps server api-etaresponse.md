# EtaResponse

An object that contains an array of one or more estimated times of arrival (ETAs).

Apple Maps Server API 1.2+

``` source
object EtaResponse
```

## Properties

`etas`

`[`EtaResponse.Eta`]`

An array of one or more EtaResponse.Eta objects.
