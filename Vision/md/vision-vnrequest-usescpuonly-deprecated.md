# usesCPUOnly

A Boolean signifying that the Vision request should execute exclusively on the CPU.

iOS 11.0–17.0DeprecatediPadOS 11.0–17.0DeprecatedMac Catalyst 13.1–17.0DeprecatedmacOS 10.13–14.0DeprecatedtvOS 11.0–17.0DeprecatedvisionOS 1.0–1.0Deprecated

``` source
var usesCPUOnly: Bool { get set }
```

## Discussion

This value defaults to false to signify that the Vision request is free to leverage the GPU to accelerate its processing.
