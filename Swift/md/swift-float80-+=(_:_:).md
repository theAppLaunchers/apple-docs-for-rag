Operator

# +=(\_:\_:)

Adds two values and stores the result in the left-hand-side variable, rounded to a representable value.

macOS 10.10+

``` source
static func += (lhs: inout Float80, rhs: Float80)
```

## Parameters 

`lhs`  

The first value to add.

`rhs`  

The second value to add.
