# isEmojiModifierBase

A Boolean value indicating whether the scalar is one whose appearance can be changed by an emoji modifier that follows it.

iOS 10.2+iPadOS 10.2+Mac Catalyst 10.2+macOS 10.12.2+tvOS 10.1+visionOS 1.0+watchOS 3.1.1+

``` source
var isEmojiModifierBase: Bool { get }
```

## Discussion

This property corresponds to the “Emoji_Modifier_Base” property in the Unicode Standard.
