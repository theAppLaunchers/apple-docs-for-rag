# -(\_:)

Calculates the additive inverse of a value.

macOS 10.10+

``` source
static func - (x: Float80) -> Float80
```

## Discussion

The unary minus operator (prefix `-`) calculates the negation of its operand. The result is always exact.

```
let x = 21.5
let y = -x
// y == -21.5
```
