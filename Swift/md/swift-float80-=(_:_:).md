# /=(\_:\_:)

Divides the first value by the second and stores the quotient in the left-hand-side variable, rounding to a representable value.

macOS 10.10+

``` source
static func /= (lhs: inout Float80, rhs: Float80)
```

## Parameters 

`lhs`  

The value to divide.

`rhs`  

The value to divide `lhs` by.
