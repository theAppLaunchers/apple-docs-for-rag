# translate(\_:)

Presents a translation of the selected text.

iOS 17.4+iPadOS 17.4+tvOS 17.4+visionOS 1.1+

``` source
optional func translate(_ sender: Any?)
```

## Parameters 

`sender`  

The object calling this method.

## Overview

To present the standard operating system UI for translating text, call translate(text:from:) in your implementation of this method.
