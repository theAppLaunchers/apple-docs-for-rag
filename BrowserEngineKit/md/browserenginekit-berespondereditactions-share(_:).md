# share(\_:)

Presents UI for sharing the selected text.

iOS 17.4+iPadOS 17.4+tvOS 17.4+visionOS 1.1+

``` source
optional func share(_ sender: Any?)
```

## Parameters 

`sender`  

The object calling this method.

## Overview

To present the standard operating system UI for sharing content, call share(text:from:) in your implementation of this method.
