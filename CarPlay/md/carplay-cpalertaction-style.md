# style

The display style for the action button.

iOS 12.0+iPadOS 12.0+Mac Catalyst 13.1+

``` source
var style: CPAlertAction.Style { get }
```

## Discussion

The combination of style and the template type, CPActionSheetTemplate or CPAlertTemplate, determines the appearance of the action button.
