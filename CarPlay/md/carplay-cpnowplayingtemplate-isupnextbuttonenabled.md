# isUpNextButtonEnabled

A Boolean value that manages the display of the Up Next button.

iOS 14.0+iPadOS 14.0+Mac Catalyst 14.0+

``` source
var isUpNextButtonEnabled: Bool { get set }
```

## Discussion

Set this property to true to have the Now Playing template display an Up Next button in the upper-right corner of the CarPlay screen.

The default value is false.
