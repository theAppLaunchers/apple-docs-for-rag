# CPNavigationAlertMinimumDuration

A constant that defines the minimum amount of time that an alert is visible.

iOS 12.0+iPadOS 12.0+Mac Catalyst 14.0+

``` source
let CPNavigationAlertMinimumDuration: TimeInterval
```

## Discussion

The minimum duration is in effect when duration is not zero.

## See Also

### Getting the Alert Duration

var duration: TimeInterval
