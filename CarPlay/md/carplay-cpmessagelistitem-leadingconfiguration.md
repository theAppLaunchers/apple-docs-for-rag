# leadingConfiguration

The configuration of the list item’s leading region.

iOS 14.0+iPadOS 14.0+Mac Catalyst 14.0+

``` source
var leadingConfiguration: CPMessageListItemLeadingConfiguration { get set }
```

## Discussion

An instance of CPMessageListItemLeadingConfiguration that describes the visual elements of the list item’s leading region. The region can show a CPMessageLeadingItem, an image, and an unread indicator.
