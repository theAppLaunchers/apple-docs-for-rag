# init(handler:)

Creates a new map button.

iOS 12.0+iPadOS 12.0+Mac Catalyst 13.1+

``` source
init(handler: ((CPMapButton) -> Void)? = nil)
```

## Parameters 

`handler`  

The block invoked after the user taps the map button.

## Return Value

A newly initialized map button.
