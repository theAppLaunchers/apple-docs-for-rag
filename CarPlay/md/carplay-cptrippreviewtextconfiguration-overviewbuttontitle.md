# overviewButtonTitle

The title displayed on the overview button.

iOS 12.0+iPadOS 12.0+Mac Catalyst 13.1+

``` source
var overviewButtonTitle: String? { get }
```

## Discussion

When the title is `nil`, the button displays a system-provided title.
