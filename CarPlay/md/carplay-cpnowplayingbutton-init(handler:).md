# init(handler:)

Creates a Now Playing button that invokes a handler.

iOS 14.0+iPadOS 14.0+Mac Catalyst 14.0+

``` source
init(handler: ((CPNowPlayingButton) -> Void)? = nil)
```

## Parameters 

`handler`  

A closure that the button invokes when the user taps it.
