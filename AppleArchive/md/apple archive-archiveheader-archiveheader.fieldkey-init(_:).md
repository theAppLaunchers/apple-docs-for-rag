# init(\_:)

Creates a new field key from the key you specify.

iOS 14.0+iPadOS 14.0+Mac CatalystmacOS 11.0+tvOS 14.0+visionOSwatchOS 7.0+

``` source
init(_ key: String)
```

## Parameters 

`key`  

The string for the key that must have a length of 3 characters that are in the range `32...127`.
