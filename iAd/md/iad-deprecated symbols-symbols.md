# Deprecated Symbols

Reference symbols from the legacy iAd framework.

## Overview

Important

iAd was a legacy framework that integrates with Apple Search Ads campaigns. iAd technology is deprecated in macOS and iOS. Use the AdServices framework for current attribution integration with the Apple Search Ads Campaign Management API.
