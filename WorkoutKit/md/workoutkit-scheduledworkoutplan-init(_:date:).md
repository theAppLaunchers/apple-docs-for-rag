# init(\_:date:)

Creates a new scheduled workout plan.

iOS 17.0+iPadOS 17.0+Mac Catalyst 18.0+macOS 15.0+watchOS 10.0+

``` source
init(
    _ plan: WorkoutPlan,
    date: DateComponents
)
```

## Parameters 

`plan`  

The target workout plan to schedule.

`date`  

Date components that determine when the workout should begin.
