# init(\_:step:)

Creates a new interval step.

iOS 17.0+iPadOS 17.0+Mac Catalyst 18.0+macOS 15.0+watchOS 10.0+

``` source
init(
    _ purpose: IntervalStep.Purpose,
    step: WorkoutStep
)
```

## Parameters 

`purpose`  

The purpose of this interval step.

`step`  

The workout step to perform.
