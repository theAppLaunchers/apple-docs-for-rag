# XOR (RealityKit)

Returns true if only one of the inputs is true.

## Parameter Types

| Input  | Type |
|--------|------|
| `In 1` | Bool |
| `In 2` | Bool |

| Output | Type |
|--------|------|
| `Out`  | Bool |
