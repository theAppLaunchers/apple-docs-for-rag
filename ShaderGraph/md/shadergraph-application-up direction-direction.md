# Up Direction

The direction of the up vector.

## Parameter Types

| Input   | Type   |
|---------|--------|
| `Space` | String |

| Output | Type     |
|--------|----------|
| `Out`  | Vector3f |

## Parameter description

`Space`  
The space for which the node returns the up vector. Defaults to “world”.
