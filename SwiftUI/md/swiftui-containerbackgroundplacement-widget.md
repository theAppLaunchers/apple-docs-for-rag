# widget

The container background placement for a widget.

iOS 17.0+iPadOS 17.0+Mac Catalyst 17.0+macOS 14.0+watchOS 10.0+

``` source
static let widget: ContainerBackgroundPlacement
```

## Discussion

Pass the container background placement to the containerBackground(_:for:) function to configure the background of your widgets.
