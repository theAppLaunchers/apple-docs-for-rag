# bottom

A point that’s centered horizontally on the bottom edge of a view.

iOS 13.0+iPadOS 13.0+Mac Catalyst 13.0+macOS 10.15+tvOS 13.0+visionOS 1.0+watchOS 6.0+

``` source
static let bottom: UnitPoint
```

## Discussion

This point occupies the position where the horizontal and vertical alignment guides intersect for bottom alignment.
