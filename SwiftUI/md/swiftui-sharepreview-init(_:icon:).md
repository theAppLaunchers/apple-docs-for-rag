# init(\_:icon:)

Creates a preview representation.

iOS 16.0+iPadOS 16.0+Mac Catalyst 16.0+macOS 13.0+visionOS 1.0+watchOS 9.0+

``` source

    _ title: Text,

```

Available when `Image` is `Never` and `Icon` conforms to `Transferable`.
