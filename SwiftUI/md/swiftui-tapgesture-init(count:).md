# init(count:)

Creates a tap gesture with the number of required taps.

iOS 13.0+iPadOS 13.0+Mac Catalyst 13.0+macOS 10.15+tvOS 16.0+visionOS 1.0+watchOS 6.0+

``` source
init(count: Int = 1)
```

## Parameters 

`count`  

The required number of taps to complete the tap gesture.
