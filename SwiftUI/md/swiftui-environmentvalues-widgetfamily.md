# widgetFamily

The template of the widget — small, medium, or large.

iOS 14.0+iPadOS 14.0+Mac Catalyst 14.0+macOS 11.0+watchOS 9.0+


## Overview

Use this value to retrieve the widget size that the user chose for a widget.
