# init(\_:\_:)

Creates a sequence gesture with two gestures.

iOS 13.0+iPadOS 13.0+Mac Catalyst 13.0+macOS 10.15+tvOS 13.0+visionOS 1.0+watchOS 6.0+

``` source
init(
    _ first: First,
    _ second: Second
)
```

## Parameters 

`first`  

The first gesture of the sequence.

`second`  

The second gesture of the sequence.
