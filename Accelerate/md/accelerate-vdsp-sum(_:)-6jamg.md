# sum(\_:)

Returns the double-precision vector sum.

iOS 13.0+iPadOS 13.0+Mac CatalystmacOS 10.15+tvOS 13.0+visionOSwatchOS 6.0+

``` source
static func sum(_ vector: U) -> Double where U : AccelerateBuffer, U.Element == Double
```

## Parameters 

`vector`  

The vector to sum.

## Discussion

This function calculates and returns the sum of the elements of a supplied vector.

The following is an example of using sum(_:):

```
let a: [Double] = [-1.5, 2.25, 3.6,
                   0.2, -0.1, -4.3]

let sum = vDSP.sum(a)

// Prints "sum 0.1500"
print(String(format: "sum %.4f", sum))
```
