# convertElements(of:to:)

Converts 8-bit signed integers to double-precision values.

iOS 13.0+iPadOS 13.0+Mac CatalystmacOS 10.15+tvOS 13.0+visionOSwatchOS 6.0+

``` source
static func convertElements(
    of source: U,
    to destination: inout V
) where U : AccelerateBuffer, V : AccelerateMutableBuffer, U.Element == Int8, V.Element == Double
```

## Parameters 

`source`  

The source vector.

`destination`  

On output, the source values converted to double-precision values.
