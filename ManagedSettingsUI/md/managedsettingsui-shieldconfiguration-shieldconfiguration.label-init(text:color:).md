# init(text:color:)

Creates a shield label with the provided text and color.

iOS 15.0+iPadOS 15.0+Mac Catalyst 15.0+

``` source
init(
    text: String,
    color: UIColor
)
```

## Parameters 

`text`  

The text to display in the label.

`color`  

The color for the text in the label.
