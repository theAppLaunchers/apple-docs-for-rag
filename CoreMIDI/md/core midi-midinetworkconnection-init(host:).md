# init(host:)

Creates a connection to the specified host.

iOS 4.2+iPadOS 4.2+Mac Catalyst 13.1+macOS 10.15+visionOS 1.0+

``` source
convenience init(host: MIDINetworkHost)
```

## Parameters 

`host`  

The host with which to establish a connection.
