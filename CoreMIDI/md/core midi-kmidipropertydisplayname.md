# kMIDIPropertyDisplayName

The user-visible name for an endpoint that combines the device and endpoint names.

iOS 4.2+iPadOS 4.2+Mac Catalyst 13.1+macOS 10.4+visionOS 1.0+

``` source
let kMIDIPropertyDisplayName: CFString
```

## Discussion

For objects other than endpoints, the display name is the same as its kMIDIPropertyName value.
