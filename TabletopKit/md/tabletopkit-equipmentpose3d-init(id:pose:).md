# init(id:pose:)

Creates a position and orientation on the table for a specific piece of equipment.

visionOS 2.0+

``` source
init(
    id: EquipmentIdentifier,
    pose: Pose3D
)
```

## Parameters 

`id`  

A unique identifier for the equipment.

`pose`  

The position and orientation of the equipment on the table.
