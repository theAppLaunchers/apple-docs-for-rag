# Rating Style Values

The size of the star image used for a rating element.

## Overview

```
NSString *const TVRatingStyleStarSmall;
NSString *const TVRatingStyleStarMedium;
NSString *const TVRatingStyleStarLarge;
```

## Topics

### Constants

TVRatingStyleStarSmall

A small star image is used.

TVRatingStyleStarMedium

A medium star image is used.

TVRatingStyleStarLarge

A large star image is used.
