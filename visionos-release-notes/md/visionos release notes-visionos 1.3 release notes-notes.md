# visionOS 1.3 Release Notes

Update your apps to use new features, and test your apps against API changes.

## Overview

The visionOS 1.3 RC SDK provides support for developing apps for Apple Vision Pro devices running visionOS 1.3. The SDK comes bundled with Xcode 15.4, available from the Mac App Store. For information on the compatibility requirements for Xcode 15.4, see Xcode 15.4 Release Notes.

### General

There are no new release notes for this software update.
