# kCFBundleNameKey

The human-readable name of the bundle.

iOSiPadOSMac CatalystmacOStvOSvisionOSwatchOS

``` source
let kCFBundleNameKey: CFString!
```

## Discussion

This key is often found in the `InfoPlist.strings` since it is usually localized.
