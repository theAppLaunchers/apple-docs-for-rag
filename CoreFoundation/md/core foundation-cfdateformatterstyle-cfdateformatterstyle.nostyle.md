# CFDateFormatterStyle.noStyle

Specifies no output.

iOSiPadOSMac CatalystmacOStvOSvisionOSwatchOS

``` source
case noStyle
```

## Discussion

You use this constant to suppress output for the date or time (see CFDateFormatterCreate(_:_:_:_:) for more details).
