# Boolean Values

CFBoolean evaluates to either true or false values where `kCFBooleanTrue` is the true, and `kCFBooleanFalse` is the false value.

## Topics

### Constants

let kCFBooleanTrue: CFBoolean!

Boolean true value.

let kCFBooleanFalse: CFBoolean!

Boolean false value.
