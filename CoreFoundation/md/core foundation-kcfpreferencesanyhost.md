# kCFPreferencesAnyHost

Indicates a preference that applies to any host.

iOSiPadOSMac CatalystmacOStvOSvisionOSwatchOS

``` source
let kCFPreferencesAnyHost: CFString
```

## Discussion

This option is not supported.
