# Token Field

Provide a text field whose text can be rendered in a visually distinct way so that users can recognize portions more easily.

## Topics

### View

class NSTokenField

A text field that converts text into visually distinct tokens.

### Management

protocol NSTokenFieldDelegate

A set of optional methods implemented by delegates of NSTokenField objects.

### Cell

class NSTokenFieldCell

A text field cell subclass that enables tokenized editing of an array of objects.

protocol NSTokenFieldCellDelegate

A set of optional methods implemented by delegates of NSTokenFieldCell objects to work with tokenized strings.
