# imageAlignment

The alignment of the receiver’s image relative to its frame.

macOS

``` source
@MainActor
var imageAlignment: NSImageAlignment { get set }
```

## Discussion

For a list of possible values, see NSImageAlignment. The default value is `NSImageAlignCenter`.
