# knobProportion

The proportion of the knob slot that the knob should fill.

macOS

``` source
@MainActor
var knobProportion: CGFloat { get set }
```

## Discussion

This property contains a floating-point value from 0.0 (minimal size) to 1.0 (fills the slot).
