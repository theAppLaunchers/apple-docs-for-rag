# NSControlGlyph

The reserved code for a control glyph.

macOS

``` source
var NSControlGlyph: Int { get }
```

## Discussion

An NSGlyphGenerator objects generates NSControlGlyph for all characters in the Unicode General Category C\* and U200B (ZERO WIDTH SPACE).
