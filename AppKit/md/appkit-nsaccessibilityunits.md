# NSAccessibilityUnits

Values that indicate the unit values of a ruler or layout area.

macOS 10.10+

``` source
enum NSAccessibilityUnits
```

## Overview

This constant is used with the accessibilityHorizontalUnits, accessibilityVerticalUnits, and accessibilityUnits properties.

## Topics

### Constants

case unknown

The units are unknown.

case inches

The units are inches.

case centimeters

The units are centimeters.

case points

The units are points.

case picas

The units are picas.
