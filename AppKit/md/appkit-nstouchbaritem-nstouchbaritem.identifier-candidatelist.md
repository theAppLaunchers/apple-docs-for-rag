# candidateList

The standard identifier for a candidate list bar item.

macOS 10.12.2+

``` source
static let candidateList: NSTouchBarItem.Identifier
```

## Discussion

This value is the identifier for NSCandidateListTouchBarItem. Use a candidate list bar item to show word completion candidates to assist users typing text.
