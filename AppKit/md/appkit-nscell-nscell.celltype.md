# NSCell.CellType

Constants for specifying how a cell represents its data (as text or as an image).

macOS

``` source
enum CellType
```

## Topics

### Constants

case nullCellType

Cell displays nothing.

case textCellType

Cell displays text.

case imageCellType

Cell displays images.
