# NSWindowDidChangeBackingPropertiesNotification User Info Properties

These constants are values that are returned in the userInfo dictionary of the didChangeBackingPropertiesNotification.

## Topics

### Constants

class let oldScaleFactorUserInfoKey: String

An NSNumber containing the old scale factor.

class let oldColorSpaceUserInfoKey: String

An NSColorSpace instance containing the old colorspace.
