# isItemExpanded(\_:)

Returns a Boolean value that indicates whether a given item is expanded.

macOS

``` source
@MainActor
func isItemExpanded(_ item: Any?) -> Bool
```

## Parameters 

`item`  

An item in the receiver.

## Return Value

true if `item` is expanded, otherwise false.
