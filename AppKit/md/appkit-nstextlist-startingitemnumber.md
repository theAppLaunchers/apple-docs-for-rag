# startingItemNumber

Sets the starting item number for the text list.

macOS 10.0+

``` source
var startingItemNumber: Int { get set }
```

## Parameters 

`itemNum`  

The item number.

## Discussion

The default value is `1`. This value will be used only for ordered lists, and ignored in other cases.
