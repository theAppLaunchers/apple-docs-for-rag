# LinearPCMFormatFlagIsPacked

A flag to indicate whether PCM sample bits use all available bits of the channel.

DriverKit 21.0+

``` source
LinearPCMFormatFlagIsPacked
```

## Discussion

This flag is a synonym if FormatFlagIsPacked.
