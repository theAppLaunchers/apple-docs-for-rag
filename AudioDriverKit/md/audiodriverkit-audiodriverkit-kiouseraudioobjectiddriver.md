# kIOUserAudioObjectIDDriver

The audio object ID of the driver.

DriverKit 21.0+

``` source
constexpr const IOUserAudioObjectID kIOUserAudioObjectIDDriver;
```

## Discussion

This IOUserAudioObjectID always refers to the one and only instance of the IOUserAudioDriver.
