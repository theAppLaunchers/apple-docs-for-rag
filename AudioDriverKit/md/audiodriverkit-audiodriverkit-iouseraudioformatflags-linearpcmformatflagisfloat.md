# LinearPCMFormatFlagIsFloat

A flag to indicate whether PCM samples are floating-point values.

DriverKit 21.0+

``` source
LinearPCMFormatFlagIsFloat
```

## Discussion

This flag is a synonym of FormatFlagIsFloat.
