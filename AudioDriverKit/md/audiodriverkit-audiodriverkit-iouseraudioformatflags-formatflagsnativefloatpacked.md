# FormatFlagsNativeFloatPacked

A flag indicating native-endian packed floating-point values.

DriverKit 21.0+

``` source
FormatFlagsNativeFloatPacked
```

## Discussion

This convienence flag set the FormatFlagIsFloat, FormatFlagsNativeEndian, and FormatFlagIsPacked flags.
