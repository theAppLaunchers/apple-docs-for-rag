# init

Initializes an empty object.

DriverKit 21.0+

``` source
bool init();
```

## Return Value

A Boolean value that indicates the result of initialization: `true` if initialization succeeded, `false` otherwise.

## Discussion

This no-arg initializer always returns `false`.
