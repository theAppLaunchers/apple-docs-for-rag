# HighPassFilterControl

The identifier for the audio high pass filter control class.

DriverKit 21.0+

``` source
HighPassFilterControl
```

## Discussion

This class is a subclass of the IOUserAudioSelectorControl class that indicates the setting of the high pass filter on the element.
