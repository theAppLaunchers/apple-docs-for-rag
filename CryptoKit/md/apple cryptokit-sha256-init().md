## Discussion

Initialize a new hash function by calling this method if you want to hash data iteratively, such as when you donâ€™t have a buffer large enough to hold all the data at once. Provide data blocks to the hash function using the `update(data:)` or update(bufferPointer:) method. After providing all the data, call finalize() to get the digest.

If your data fits into a single buffer, you can use the `hash(data:)` method instead, to compute the digest in a single call.
