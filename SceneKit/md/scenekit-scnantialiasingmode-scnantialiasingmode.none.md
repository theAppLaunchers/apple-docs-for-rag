# SCNAntialiasingMode.none

Disables antialiased rendering.

iOSiPadOSMac Catalyst 13.1+macOS 10.10+tvOSvisionOSwatchOS

``` source
case none
```

## Discussion

This is the default antialiasing mode on iOS.
